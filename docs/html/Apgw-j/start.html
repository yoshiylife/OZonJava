アプリケーションゲートウェイの起動

アプリケーションゲートウェイの起動には次の条件が満たされていなければならない。

  • 環境変数 AGHOME が適切に設定されている。
  • 起動する計算機に Java の実行環境(JDK)がインストールされている。
  • 環境変数 CLASSPATH にアプリケーションゲートウェイのクラス ファイル $AGHOME/lib/ApplicationGateway.zip が含まれるように設定されている。
起動方法は、新たにウィンドウを作成して $AGHOME/bin/ag[.bat] を起動する(スクリプトである)。 これはアプリケーションゲートウェイは標準入力からコマンドを 受け付けるようになっているからである。
アプリケーションゲートウェイの終了は exit コマンドによって行う。
アプリケーションゲートウェイを介してリモートメソッド起動を 行うためには、双方のサイトでアプリケーションゲートウェイ、 OZ ドメインリゾルバ、OZ ホームリゾルバが稼働している必要が有る。
起動の順序は OZ ドメインリゾルバが先で OZ ホームリゾルバが後である。
