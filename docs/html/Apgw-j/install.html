アプリケーションゲートウェイのインストール

アプリケーションゲートウェイは、アプリケーションゲートウェイ 本体と CGI プログラムが必要である。
アプリケーションゲートウェイは Java プログラムであり、OzVM とは独立である。
アプリケーションゲートウェイは、環境変数 AGHOME で 指定されるディレクトリをワークディレクトリに用いる。 ワークディレクトリには、設定用ファイルを格納するディレクトリ"etc"、 実行用ファイルを格納するディレクトリ"bin"およびログを格納する ディレクトリ"logs"がある("logs"は初めてアプリケーションゲートウェイが 起動される際に自動的に生成される)。
パッケージは jp.go.ipa.oz.system であり、クラスファイルは $AGHOME/lib/ApplicationGateway.zip を用いる。
配布パッケージでは、ワークディレクトリのイメージを $OZHOME/ag に 作成してある。 ワークディレクトリが OZHOME のサブディレクトリである必要はないので、 適当なディレクトリにコピーしてかまわない。
例えば、$OZHOME/ag を /etc/ag としてコピーした場合、環境変数 AGHOME を /etc/ag に設定する。 但し、OZHOME と AGHOME を一致させると、ログファイルを格納するディレクトリが共通になるので避けたほうがよい。
