アプリケーションゲートウェイのコマンド

アプリケーションゲートウェイのコマンドとして次のものを提供している。

  • exit
    アプリケーションゲートウェイの終了
  • domainlist
    ドメインの一覧表示
  • domain domain-name
    ドメインを中継する CGI の URL を表示
  • update
    ドメイン表をマスターファイルからロードして更新
  • accesslist
    アクセス制限の一覧表示
  • enable <ドメイン名もしくは OZ ホーム名>
    ドメインもしくは OZ ホームとのリモートメソッド起動を可能にする
  • disable <ドメイン名もしくは OZ ホーム名>
    ドメインもしくは OZ ホームとのリモートメソッド起動を拒絶するようにする
  • help
    コマンド一覧表示
