分散システム統合開発環境


ここでは、OZ分散システム統合開発環境( 以下、OZ IDE と略します。)の使用方法をご紹介しています。OZ IDE はOZアプリケーションの構築を支援するツールです。


OZ IDE を利用すると、以下の機能を利用することができます。

１．用途に合わせた OZ ソースファイルのコンパイルを行うことができます。
インタフェースコンパイル、クラスコンパイルだけではなく、自動コンパイル、リリース用コンパイルを行うことができます。

２．開発に必要なクラスライブラリを閲覧することができます。
クラスメンバをフラットに知ることができます。そのクラスで宣言されているメンバだけではなく上位クラスから継承するメンバも同時に閲覧できるだけではなく、メソッド本体定義、継承関係などクラスに関する情報を総合的に知ることができます。

３．構築したOZアプリケーションのリリースパッケージを作成することができます。
OZアプリケーションはZIP形式でリリースします。OZ IDEはリリース単位でZIP形式ファイルを作成する機能を持っています。

４．カタログサーバへのインポート、エクスポートができます。
カタログサーバはOZアプリケーションやクラスライブラリを貯蔵するサーバです。OZ IDEを利用すると、カタログサーバを閲覧し、そこにカタログされているOZアプリケーション、クラスライブラリをインポートしたり、そこにリリースパッケージをエクスポートしたりすることができます。


OZ IDE は、以下のツールから構成されています。

  • プロジェクトマネージャ
  • スクールブラウザ
  • クラスブラウザ
  • サブジェクトカタログブラウザ


OZ IDE を起動すると、プロジェクトマネージャとスクールブラウザが表示されます。この2つのウィンドウは対になって動作します。

クラスブラウザから、クラスの継承構造を図示する継承ブラウザを起動することができます。
