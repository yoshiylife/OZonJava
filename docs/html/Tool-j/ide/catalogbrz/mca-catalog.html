サブジェクトカタログブラウザのCatalogメニュー


サブジェクトカタログブラウザの Catalog メニューには、input GOL と contact to trader のメニュー項目があります。



input GOL ブラウズするカタログサーバを切り替えます。
複数のカタログサーバに登録されたカタログを閲覧することができます。
contact to trader カタログサーバに登録されたカタログをインポートします。
現在は公開されていません。
