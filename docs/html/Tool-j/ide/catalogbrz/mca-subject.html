サブジェクトカタログブラウザのSubjectメニュー


サブジェクトカタログブラウザの Subject メニューには、show summary、import、remove のメニュー項目があります。



show summary カタログサーバにカタログされたサブジェクトの説明を参照します。
import カタログサーバにカタログされたサブジェクトをインポートします。
remove カタログサーバにカタログされたサブジェクトを消去します。
