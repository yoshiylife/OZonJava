カタログを削除する


サブジェクトカタログブラウザから、カタログサーバに登録されたカタログを削除することができます。




１．カタログブラウザの subject list: メニューリストから削除するサブジェクトを選択してください。

２．Subject メニューから remove メニュー項目を選択してください。


カタログサーバからそのサブジェクトが削除されます。そして、subject list: メニューリストからそのエントリが消去されます。
