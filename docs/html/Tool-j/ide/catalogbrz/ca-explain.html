カタログの説明を照会する


カタログサーバに登録されたカタログにはインポート時に説明を書き込むことができます。カタログをエクスポートする時には、どのようなカタログなのかをその説明によって知ることができます。





１．サブジェクトカタログブラウザの subject list: メニューリストからカタログを選択してください。

２．サブジェクトカタログブラウザの Subject メニューから show summary メニュー項目を選択してください。


サブジェクトブラウザが開かれ、サブジェクト名とともにその説明が表示されます。
