カタログをIDEにインポートする


カタログサーバに登録されたカタログされたサブジェクトをスクールブラウザにインポートすることができます。




１．カタログブラウザの subject list: メニューリストからインポートするサブジェクトを選択してください。

２．カタログブラウザの Subject メニューから import メニュー項目を選択してください。


スクールブラウザにそのサブジェクトが追加されます。
