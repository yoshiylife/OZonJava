サブジェクトカタログブラウザの構成


OZ IDE のスクールブラウザの Catalog メニューから import subject... メニュー項目を選択すると、サブジェクトカタログブラウザが開かれます。



サブジェクトカタログブラウザのメニューバー

System
 サブジェクトカタログブラウザを再表示、終了に関するコマンド
Subject
 カタログに関するコマンド
Catalog
 カタログブラウザに表示するサーバ選択のコマンド



サブジェクトカタログブラウザの表示エリア

server GOL:
 サブジェクトカタログサーバの GOL が表示されます。
subject list:
 サブジェクトカタログサーバに登録されているサブジェクトが一覧されます。
