サブジェクトカタログブラウザを開く


サブジェクトカタログブラウザをOZ IDEから起動する場合には、スクールブラウザから行います。




スクールブラウザの Catalog メニューから import subject... メニュー項目を選択してください。


サブジェクトカタログブラウザが開きます。



はじめてカタログブラウザを開く場合には、ブラウズしたいカタログサーバのGOLを入力するためのダイアログが開かれます。

カタログサーバのGOLをキー入力しOK ボタンをクリックすると、Subject list: にサブジェクトがリストされます。
Cancelボタンをクリックすると、このダイアログは消去されます。この場合にはブラウザには何もブラウズされません。Catalogメニューのcontactメニュー項目によってカタログサーバを指定することができます。また、指定したカタログサーバが起動されていないとサブジェクトのリストは行われません。
