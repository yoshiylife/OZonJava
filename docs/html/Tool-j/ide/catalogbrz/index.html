カタログブラウザ


開発者が共用するOZアプリケーションやクラスライブラリをサブジェクト単位でカタログサーバに登録することができます。各開発者はそれを閲覧し、自分の開発環境(IDE)に必要となるカタログをサブジェクトとしてインポートすることができます。サブジェクトカタログブラウザはカタログサーバが管理するカタログをブラウズする機能、および、開発環境へのインポートする機能を持っています。アプリケーションカタログブラウザを利用すると実行可能なカタログを登録することができます。

カタログブラウザはスクールブラウザから起動することができます。






サブジェクトカタログブラウザとアプリケーションカタログブラウザの機能と操作は類似しています。ここではサブジェクトカタログブラウザに使用方法をご紹介しています。


  1. カタログブラウザの構成要素
    1. サブジェクトカタログブラウザの構成
    2. サブジェクトカタログブラウザのSystemメニュー
    3. サブジェクトカタログブラウザのSubjectメニュー
    4. サブジェクトカタログブラウザのCatalogメニュー

  2. カタログブラウザの起動と終了
    1. サブジェクトカタログブラウザを開く
    2. サブジェクトカタログブラウザを閉じる

  3. カタログブラウザの使用方法
    1. カタログを閲覧する
    2. カタログの説明を照会する
    3. カタログをIDEにインポートする
    4. カタログを削除する
    5. カタログサーバを切り替える
    6. カタログを再表示する
