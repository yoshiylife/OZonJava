サブジェクトを登録する


スクールブラウザはサブジェクト単位でクラスを管理します。必要になったサブジェクトをスクールブラウザに登録して作業を行います。サブジェクトはアプリケーションのリリース単位です。大部分のサブジェクトは他のサブジェクトのクラスを利用して構築され活動します。

プロジェクトマネージャでは、サブジェクト単位でコンパイルを行うことができます。



１．スクールブラウザの School メニューから add subject メニュー項目を選択します。

サブジェクトを登録するためのダイアログが表示されます。



２．サブジェクト名を入力しOKボタンをクリックすると、スクールブラウザにそのサブジェクトが登録され、Subject メニューリストにそのサブジェクトが追加されます。
