スクールをエクスポートする


スクールブラウザが管理するサブジェクトのスクールを、ファイルに保存します。


１．スクールブラウザのウィンドウ内に存在する Subject:メニューリストから、エクスポートするサブジェクトを選択します。


２．スクールブラウザの File メニューから export メニュー項目を選択してください。

スクールを保存するファイルを入力するためのウィンドウが表示されます。

３．ファイル名(N):入力エリアにスクールファイル名をキーボード入力し、保存ボタンをクリックするとスクールファイルが作成保存されます。

エクスポートされたスクールファイルの例
"Quit","sample.Quit_if","sample.Quit_cl"
"Test","sample.Test_if","sample.Test_cl"
"MyDialog","sample.MyDialog_if","sample.MyDialog_cl"
