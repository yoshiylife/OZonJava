サブジェクトの名前を変更する


スクールブラウザが管理するサブジェクトの名前を変更することができます。



１．スクールブラウザの School メニューから rename subject メニュー項目を選択します。

サブジェクト名を変更するためのダイアログが表示されます。



２．ダイアログの下方のメニューリストにはサブジェクトがリストされます。(lib と ide は名前を変更することができません。) その中から名前を変更したいサブジェクトを選択してください。そして上方の入力エリアに新規サブジェクト名を入力して、OKボタンをクリックしてください。指定したサブジェクトの名前が変更されます。

スクールブラウザのウィンドウ内に存在する Subject メニューリストでサブジェクト名の変更を確認することができます。
