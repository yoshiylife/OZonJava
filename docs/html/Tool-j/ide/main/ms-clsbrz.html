スクールブラウザのClassBrowserメニュー



スクールブラウザのSchool メニューには、ClassBrowser メニューには、open (selected label) のメニュー項目があり、サブメニュー項目として、intereface part、implementation part を用意しています。



intereface part.. 選択されているクラスのインタフェースを表示するクラスブラウザを開きます。
implementation part. 選択されているクラス(動作部)を表示するクラスブラウザを開きます。
