プロジェクトマネージャのProjectメニュー


プロジェクトマネージャの Project メニューには、new、open、save、save as、remove、endのメニュー項目があります。



new 新規プロジェクトを作成します。
open... 既存プロジェクトを開きます。
save カレントプロジェクトを保存します。
save as... プロジェクトを別名で保存します。
remove... 既存プロジェクトを削除します。
end OZ IDEを終了します。
