サブジェクトに登録されたクラス名/クラス識別子を変更する


登録時に設定されたクラスの名前および識別子を変更することができます。



１．スクールブラウザの左側に存在するLabel:リストからクラスの情報を変更するクラスを選択してください。


２．スクールブラウザの School メニューから rename class メニュー項目を選択してください。


クラス名/クラス識別子を変更するためのダイアログが表示されます。


rename class メニュー項目が選択されたときに左のリストでマーキングされていたクラスのクラス名/クラス識別子が表示されています。


３．クラス名、クラスのインタフェース部識別子、クラスの動作部識別子の編集してください。すべてを変更することもできますし、その中の一部を変更することもできます。


そして、OKボタンを押下すると、クラスのクラス名/クラス識別子が変更されます。
