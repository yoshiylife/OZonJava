プロジェクトファイルを最新表示する


プロジェクトディレクトリへのファイルの追加、削除を行った場合、プロジェクトを切り替えたときには、プロジェクトファイルを最新表示してください。Project Files:リストにプロジェクトディレクトリの現在の状況が反映されます。

プロジェクトファイルを最新表示する2つの方法があります。


<方法１>プロジェクトマネージャの Edit メニューから refresh メニュー項目を選択してください。


Project Files リストが最新表示されます。



<方法２>プロジェクトマネージャのProject Files: ラベルの横にある Refresh ボタンをクリックしてください。Project Files リストが最新表示されます。
