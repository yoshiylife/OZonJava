プロジェクトマネージャの構成


プロジェクトマネージャはOZ IDEを起動すると表示されます。


プロジェクトマネージャのメニューバーには、4つのメニューがあります。

Project
プロジェクトを新規生成、開く、保存するなどを行うコマンド
Edit
プロジェクトの編集に関するコマンド
Compile
プロジェクトをコンパイル/テストするためのコマンド
Help
バージョン情報表示のためのコマンド



2つの表示エリアには、それぞれ次の情報が表示されます。

Project Directory プロジェクトの格納場所が表示されます。
Project Files プロジェクトファイルの名前が表示されます。
