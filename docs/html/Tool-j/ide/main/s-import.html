スクールをインポートする


スクールファイルをスクールブラウザに取り込みます。このスクールファイルはサブジェクトスクールだけを記述されたものであることを想定しています。標準クラスライブラリや既に取り込まれているクラスのスクールは不要です。


１．スクールブラウザのウィンドウ内に存在する Subject:メニューリストからサブジェクトを選択します。

インポートするスクールと対応付けられるサブジェクトが存在しない場合には、サブジェクトを追加してください。


２．スクールブラウザの File メニューから import メニュー項目を選択してください。

ファイルを選択するためのウィンドウが表示されます。

インポートするスクールファイルを選択して「開く」ボタンをクリックすると、そのスクールがスクールブラウザにインポートされます。インポートされたスクールは １．で選択したサブジェクトに登録されます。
