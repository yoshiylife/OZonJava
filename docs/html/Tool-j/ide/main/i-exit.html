OZ IDEを終了する


OZ IDEを終了するとは、OZ IDEのグローバルオブジェクトを非活性化することです。

OZ IDEの非活性化は、スクールブラウザのFileメニューあるいはプロジェクトマネージャのProjectメニューに存在する end メニュー項目で行います。



  1. end を選択してください。
  2. 終了を確認するメッセージダイアログが表示されますから、OKボタンをクリックしてください。それまでの作業状況が保存されて、OZ IDEオブジェクトが非活性化されます。
    Cancelボタンをクリックすると、OZ IDEは非活性化されません。作業を継続することができます。
