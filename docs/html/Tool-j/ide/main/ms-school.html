スクールブラウザのSchoolメニュー


スクールブラウザのSchool メニューには、School メニューには、add class、remove class、rename class などのメニュー項目があります。



add class... サブジェクトにクラスを登録します。
remove class... サブジェクトからクラスを削除します。
rename class... サブジェクトに登録されたクラスのスクールを変更します。
add subject... サブジェクトを登録します。
remove subject... サブジェクトを削除します。
rename subject... サブジェクトの名前を変更します。
class ID generate... クラス識別子を自動付与します。
select subject... プロジェクトで使用するサブジェクトを選択します。
