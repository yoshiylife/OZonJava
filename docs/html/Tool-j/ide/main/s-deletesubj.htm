サブジェクトを削除する


スクールブラウザはサブジェクト単位でクラスを管理します。不用になったサブジェクトをスクールブラウザから取り除くことができます。



１．スクールブラウザの School メニューから remove subject メニュー項目を選択します。

サブジェクトを削除するためのダイアログが表示されます。



２．このダイアログ中央のリストには、lib、ide 以外のサブジェクトがリストされています。その中から削除するサブジェクトを選択しOKボタンをクリックすると、そのサブジェクトがスクールブラウザから削除されます。ただし、管理するクラスが存在しないサブジェクトのみ削除できます。

空でないサブジェクトを選択して削除しようとしても削除できません。先に「サブジェクトからクラスを削除する」を実行してください。
