クラスブラウザを開く


クラスブラウザはクラスについての総合的な情報を提供するツールです。スクールブラウザからクラスブラウザを起動することができます。



１．スクールブラウザの左側に存在するLabel:リストからクラスを選択してください。



２．そして、ClassBrowser メニューからopen(selected lavel) メニュー項目を選択すると、クラスブラウザが開かれます。
interface part サブメニュー項目を選択すると、選択したクラスのインタフェース部の情報を参照できます。implement part サブメニュー項目を選択すると、選択したクラスの情報を参照できます。
