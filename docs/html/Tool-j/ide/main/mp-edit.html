プロジェクトマネージャのEditメニュー


プロジェクトマネージャの Edit メニューには、refresh、set directory、edit file、set editor のメニュー項目があります。



refresh Project Filesエリアを最新表示します。
set directory... プロジェクトディレクトリを設定します。
edit file プロジェクトファイルを編集します。
set editor... プロジェクトファイルの編集で使用する外部エディタを登録します。
