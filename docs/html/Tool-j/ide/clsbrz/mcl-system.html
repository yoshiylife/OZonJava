クラスブラウザのSystemメニュー


クラスブラウザの System メニューには、new、open、save、save as、remove、end のメニュー項目があります。



open クラスビューを新規に開きます。
クラスの照会、クラスのインタフェースの照会を行うことができます。
close そのクラスブラウザを閉じます。
close all 開かれているすべてのクラスブラウザを閉じます。
