インタフェースビュー：クラスのインタフェース情報を表示する




クラスブラウザのメニューバー

class ID 表示エリア ブラウズされているクラスのクラス識別子を表示します。
super classes(if) 表示エリア 上位クラスが存在すれば、そのクラス識別子をリスト表示します。
cell チェックボックス ブラウズされているクラスがcell型であると、ここにチェックが付けられます。
abstract チェックボックス ブラウズされているクラスが abstruct メソッドを持つクラスであると、ここにチェックが付けられます。
final チェックボックス ブラウズされているクラスが final 修飾子を持つクラスであると、ここにチェックが付けられます。
members リスト クラスの public メンバを表示しています。（OZ言語ではメソッドだけがpublicメンバです。）

各メンバについて、
  • アクセスレベル（必ずpublic）
  • メソッド修飾子
  • 返り型
  • メンバ名
  • 引数リスト
  • 定義クラス（*は、このクラスで宣言されているメンバ）
の情報が表示されます。
