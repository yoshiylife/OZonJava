クラスブラウザを開く


最初のクラスブラウザはスクールブラウザから開きます。1つ以上のクラスブラウザが開いている状態では、スクールブラウザだけでなく、クラスブラウザからも新規のブラウザを開くことができます。



スクールブラウザからクラスブラウザを開く

１．スクールブラウザのSubject:メニューリストに表示されているクラスをクリックしてください。

２．ClassBrowser メニューから interface part サブメニュー項目あるいは implementation part を選択してください。


クラスブラウザが新規に開かれます。interface part を選択した場合にはクラスのインタフェースの情報、implementation part を選択した場合にはクラスの情報を表示するクラスブラウザが開かれます。
