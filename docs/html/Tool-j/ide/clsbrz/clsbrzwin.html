クラスブラウザの構成


OZシステムのクラスはインタフェース部と動作部でユニークに識別されますが、クラスブラウザはその各々に対応するビューを持っています。
同時に複数のクラスのビューを表示することができます。

インタフェースビュー：クラスのインタフェース情報を表示するためのビュー



クラスビュー：クラスの情報を表示するためのビュー



クラスブラウザのメニューバー

System
 クラスのブラウズを開始/終了するコマンド
Hierarchy
 祖先クラスの継承関係を表示するコマンド
