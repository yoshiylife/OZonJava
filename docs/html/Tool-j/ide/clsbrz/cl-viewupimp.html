上位実装クラスを照会する


上位クラスのうち動作部を継承するクラスを上位実装クラスと呼びます。
上位実装クラスの情報は、その上位クラスの識別子をダブルクリックすることによって簡単に照会できます。





クラスの情報が表示されているスクールブラウザには、現在照会されているクラスの上位実装クラスの動作部識別子が表示されています（ super classes(if) 表示エリア）。


super class(cl) に表示されているクラス識別子をダブルクリックすると、そのクラスの情報を表示したクラスブラウザが開きます。

（注意：OzCellクラスの情報は表示されません。）
