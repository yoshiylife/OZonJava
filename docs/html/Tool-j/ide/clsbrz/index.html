クラスブラウザと継承ブラウザ


クラスブラウザはクラスに関する情報を統合的に提供するツールです。アトリビュート、メソッド、コンストラクタ、上位クラスなどクラスに関する情報をさまざまな角度から知ることができます。継承ブラウザはクラスの継承関係を図示するツールです。

クラスブラウザはスクールブラウザから起動することができます。継承ブラウザはクラスブラウザから起動することができます。
