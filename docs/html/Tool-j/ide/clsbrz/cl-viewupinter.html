上位クラスのインタフェースを照会する


上位クラスのインタフェース情報は、その上位クラスの識別子をダブルクリックすることによって簡単に照会できます。





インタフェース情報が表示されているクラスブラウザには、現在照会されているクラスの上位クラスがリストされています（ super classes(if) 表示エリア）。


super classes(if) にリストされているクラス識別子をダブルクリックすると、そのクラスのインタフェース情報を表示するクラスブラウザが開きます。
