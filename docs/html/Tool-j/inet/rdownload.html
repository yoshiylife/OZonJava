クラスのダウンロード


OZクラスの実行形式ファイルをインタネットで接続されたマシンからローカルマシンにダウンロードすることができます。



「HTML文書のカスタマイズ」での設定を行ったwwps-menu.htmlをWWWブラウザで開いて、Download ボタンをクリックしてください。



クラスＩＤ入力エリアにクラス識別子（末尾が_clの識別子）をキー入力して送信ボタンをクリックしてください。
実行形式のクラスを格納する場所を問い合わせるダイアログが表示されます。

そこで格納ディレクトリとファイル名を指定すると、クラスの実行形式ファイルがそのディレクトリにその名前でダウンロードされます。
