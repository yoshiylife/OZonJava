サーバの生成


ワールドワイドプログラミングサーバの機能を利用するためには、OZCGIサーバが必要です。２つのサーバを生成する必要があります。通常の方法でOZCGIサーバとワールドワイドプログラミングサーバを生成することができます。


OZCGIサーバのクラス識別子は、JP.go.ipa.oz.user.inter.cgi.OzCGI_clです。GODとして、「ozcgi」あるいは「OZCGI」を指定してください。

ワールドワイドプログラミングサーバIのクラス識別子は、JP.go.ipa.oz.user.inter.wwps.WWPServer_clです。


生成後、必ずサーバを停止してください。
