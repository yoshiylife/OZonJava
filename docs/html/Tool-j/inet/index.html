ワールドワイドプログラミングサーバ


OZシステムは、既存のインタネットサービスと接続するために

  1. OZCGIサーバ
  2. ワールドワイドプログラミングサーバ
  3. インタネットクラスライブラリ

を提供しています。ここでは、ワールドワイドプログラミングサーバの使用方法をご紹介しています。インタネットクラスライブラリは各種インタネットサービスへのアクセスをOZプログラム上で容易にするためのクラスライブラリで、クラスライブラリで扱っています。


OZCGIサーバ

WWWサーバを通じてOZプログラムのサービスの利用可能とするサーバです。サーバとコミュニケーションするCGI（Common Ｇａｔｅｗａｙ Intereface）プログラムも提供しています。


ワールドワイドプログラミングサーバ

WWWクライアントからOZプログラムの開発を可能にするためのサーバです。OZCGIサーバを使用して、
  • リモートコンパイル
  • リモート実行
  • OZクラスのダウンロード
  • カタログサーバの閲覧・インポート・エクスポート(現在利用できません)
のサービスを提供します。

これらのサービスを利用するためのテンプレート（HTML）も提供しています。このテンプレートをカスタマイズして利用してください。



OZ CGI サーバを利用して、ワールドワイドプログラミングサーバのようにOZアプリケーションをWWWクライアントから操作するようなヒューマンインタフェースを持つアプリケーションを構築することができます。この詳細については、「プログラミングガイド」を参照してください。
