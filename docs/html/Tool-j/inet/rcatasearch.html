リモートカタログの検索


OZCGIサーバが起動しているマシン上のカタログサーバに登録されたカタログを検索することができます。



「HTML文書のカスタマイズ」での設定を行ったwwps-menu.htmlをWWWブラウザで開いて、Component Search ボタンをクリックしてください。


  1. 「カタログ名」にアクセスしたいカタログを指定します。(カタログサーバのGODを指定してください。)

  2. 「キーワード」に検索キーワードを入力します。カタログ生成時に入力した「サブジェクト名」、「コメント」から、そのキーワードに該当するものを検索します。


上記の2つのエリアの入力を行って送信ボタンをクリックすると、WWWサーバと接続され、WWWサーバ上のカタログサーバ検索が行われます。

検索が終了すると、WWWブラウザの画面が検索結果を表示する画面に切り替わります。
開発環境を使用して登録されたカタログも検索されます。
