リモート実行


OZアプリケーションのリモート実行を行うことができます。リモート実行すると、サーバマシンでOZアプリケーションが実行され、その結果がクライアントマシンに返されます。



「HTML文書のカスタマイズ」での設定を行ったwwps-menu.htmlをWWWブラウザで開いて、Execute ボタンをクリックしてください。


  1. クラスＩＤ 入力エリアに実行するアプリケーションのクラス識別子（末尾が_cl）をキー入力してください。
    クラス識別子はフルクオリファイド形式 、ドメイン名のプレフィックス (JP.go.ipa.oz) を除いた形式 のいずれかで指定します。

  2. リスト形式のパラメータ入力エリアには、第1引数として、出力結果を格納するファイルを指定します。そのアプリケーションの引数(セルの引数)をキー入力します。
    引数は空白（スペース）で区切ってください。


上記の2つのエリアの入力を行って送信ボタンをクリックすると、WWWサーバと接続され、WWWサーバ上の OZ アプリケーションが実行されます。

リモート実行が終了すると、WWWブラウザの画面がリモート実行の標準出力を表示する画面に切り替わります。結果は代引数のファイルに格納されています。
