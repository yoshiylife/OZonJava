ランチャのウィンドウ構成


ランチャを起動すると、「OZ Launcher」というタイトルを持つウィンドウが開きます。


ランチャのメニューバーには、2つのメニューがあります。

System
ランチャ自身に関するコマンド
Cell
グローバルオブジェクトに関するコマンド



3つの表示エリアには、それぞれ次の情報が表示されます。

directories: ドットで表わされた名前を持つグローバルオブジェクトの存在を示します。
inactive cells: 非活動中のグローバルオブジェクトがリスト表示されます。
active cells: 活動中のグローバルオブジェクトがリスト表示されます。
