グローバルオブジェクトを活性化する


OZシステムでアプリケーションの実行とはグローバルオブジェクトの活性化を意味します。ランチャを利用すると、簡単にグローバルオブジェクトを活性化することができます。




１． inactive cells: リストから活性化するオブジェクトを選択してください。


２．Cell メニューから start cell メニュー項目を選択してください。

選択したオブジェクトが活性化されます。


inactive cells: リストのオブジェクトをダブルクリックするという方法でも、オブジェクトを活性化させることができます。
