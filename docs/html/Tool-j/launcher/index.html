ランチャ


ここでは、ランチャの使用方法をご紹介しています。ランチャはカタログサーバからアプリケーション、ツールを検索し、そのグローバルオブジェクトを生成、活性化するツールです。


ランチャを利用すると、以下のことを容易に行うことができます。

  1. カタログサーバに格納されたOZアプリケーションを検索しインポートできます。
  2. グローバルオブジェクトを生成します。
  3. グローバルオブジェクトの実行状況を調べることができます。
  4. グローバルオブジェクトを活性化させることができます。





  1. ランチャの構成要素
    • ランチャのウィンドウ構成
    • ランチャのSystemメニュー
    • ランチャのCellメニュー

  2. ランチャの起動と終了
    • ランチャを生成する
    • ランチャを起動する
    • ランチャを終了する

  3. ランチャの使用方法
    • グローバルオブジェクトを生成する
    • グローバルオブジェクトを活性化する
    • グローバルオブジェクトを閲覧する
    • ランチャを再表示する
