ランチャを終了する


ランチャを終了するとは、ランチャオブジェクトを非活性化することです。

ランチャの非活性化は、System メニューの ｑｕｉｔメニュー項目で行います。

１．System メニューの quit を選択してください。

２．終了を確認するメッセージダイアログが表示されますからOKボタンをクリックしてください。それまでの作業状況が保存されて、ランチャオブジェクトが非活性化されます。


Cancelボタンをクリックするとランチャは非活性化されません。作業を継続することができます。

OzVMシェルでランチャを活性化した場合には、stopコマンドによっても非活性化することができます。
