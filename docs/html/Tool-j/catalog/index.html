カタログサーバ


ここでは、カタログサーバの使用方法をご紹介しています。カタログサーバはOZアプリケーションやOZクラスライブラリを登録することができるサーバです。IDEで開発したOZアプリケーションやOZクラスライブラリは、カタログサーバを介して配布することができます。また、ランチャを利用して、OZアプリケーションの実行を行うことができます。


カタログサーバは、以下のサービスを提供します。

  1. OZアプリケーションやOZクラスライブラリを蓄積することができます。
  2. リモート環境でOZアプリケーションやOZクラスライブラリを共用することができます。





  1. カタログサーバの起動と終了
    • カタログサーバを生成する
    • カタログサーバを起動する
    • カタログサーバを終了する
  2. カタログサーバとのインタフェース
    • サブジェクトカタログブラウザ
    • アプリケーションカタログブラウザ
    • カタログサーバへのインタフェースを持つツール
  3. カタログサーバの使用方法
    • カタログをインポートする
    • カタログをエクスポートする
    • カタログを閲覧する
