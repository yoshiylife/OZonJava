カタログサーバを終了する


カタログサーバをOzVMのインタラクティブモードを利用して起動した場合には、stopコマンドで終了（＝非活性化）させてください。
ランチャから起動した場合には、以下の方法で終了させてください。

  1. セル stopper を起動してください。

    例)
    > oz stopper



  2. catalog の GOD を入力します。

  3. Stop Cell ボタンをクリックします。


  4. Quit ボタンをクリックすると、stopperのウィンドウが消去されます。


    停止処理を行わないでホストをシャットダウンした場合、再起動できないので注意してください。
