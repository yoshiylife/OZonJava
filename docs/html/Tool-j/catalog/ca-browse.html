カタログを閲覧する

アプリケーションカタログブラウザとサブジェクトカタログブラウザを利用して、カタログサーバに登録されているカタログを閲覧することができます。

  1. アプリケーションカタログブラウザには実行可能として登録されたカタログが表示されます。

  2. サブジェクトカタログブラウザには実行可能であるもの、クラスライブラリなどIDEでひとつのサブジェクトとして作成されたものがカタログとして表示されます。


また、ワールドワイドプログラミングサーバであるマシン上のカタログは、WWWブラウザで照会することができます「カタログのリモート照会」。キーワードのよる検索も行うことができます「カタログのリモート検索」。
