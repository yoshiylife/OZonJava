カタログをインポートする

アプリケーションカタログブラウザとサブジェクトカタログブラウザを利用して、カタログサーバからカタログをインポートすることができます。この両者はランチャとOZ IDEから起動されます。

  1. アプリケーションカタログブラウザには実行可能として登録されたカタログが表示されます。ランチャを利用してOZホームにインポートします。

  2. サブジェクトカタログブラウザには、実行可能であるもの、クラスライブラリなどIDEでひとつのサブジェクトとして作成されたものがカタログとして表示されます。現在活動中のOZ IDEにインポートすることができます「カタログサーバにサブジェクトをインポートする」。
