カタログサーバへのインタフェースを持つツール

以下のツールからサブジェクトカタログブラウザ、アプリケーションカタログブラウザが開かれます。

  1. OZ IDE のスクールブラウザ

    サブジェクトカタログブラウザとアプリケーションカタログブラウザを開くことができます。


  2. ランチャ

    アプリケーションカタログブラウザを開くことができます。



また、WWWブラウザを利用して、ワールドワイドプログラミングサーバであるマシン上のカタログサーバにエクスポート、閲覧、検索することができます「ワールドワイドプログラミングサーバ」。
