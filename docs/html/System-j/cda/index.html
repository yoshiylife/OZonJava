クラス配送

OZ は、ネットワークを通じてオブジェクトを送受信する機能を持つ。 このとき、受信側の OZ ホームに存在しないクラスのインスタンスを送信しても、ネットワークを通じて自動的に必要なクラスの配送を受けることにより、そのオブジェクトを動作させたいという要求がある。
クラス配送エージェント(CDA と呼ぶ)は、エグゼキュータが必要としたクラスを求めてネットワーク上を検索し、見つかったらネットワークを通してエグゼキュータがロードできるように配送する。
