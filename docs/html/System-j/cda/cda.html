クラス配送エージェント(CDA)

CDA はエグゼキュータとのクラス配送の一部通信には ozclass というプロトコルによるソケット通信を使用する。
CDA を起動する OZ ホームの設定ファイル ($OZHOME/lib/ 下)は次のようにする。

oz.properties
oz.classloader.deliveryAgent.url=ソケットのホスト名とポート番号
例えば host1 というマシンの 6010 番ポートを指定したいのなら次のような記述となる。 
oz.classloader.deliveryAgent.url=ozclass://host1:6010/
エグゼキュータから CDA へのグローバルクラスサーチ回数指定を指定することができる。
oz.classloader.deliveryAgent.retry=サーチ回数
例えば、グローバルクラスサーチ回数を二回と指定する場合は次のような記述となる。 
oz.classloader.deliveryAgent.retry=2
