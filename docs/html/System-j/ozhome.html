OZ ホームの構成

OZ ホームのサブディレクトリと内容は以下である。

表: OZ ホームディレクトリの構成
ディレクトリ名 内　　容
bin OZ システムコマンド
docs 各種ドキュメント
lib OZ ライブラリ群、設定ファイル
secure セキュリティのための設定ファイル
insecure セキュリティのためのディレクトリ
objects オブジェクトイメージディレクトリ
src 各種ソースプログラム
user サンプルプログラム、ユーザソース
ag アプリケーションゲートウェイ環境
