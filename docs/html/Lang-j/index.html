OZ言語



OZ言語の特徴

分散環境下でのアプリケーション構築を容易なものにするために、OZシステムはOZ言語という特別なプログラミング言語を備えています。OZ言語は以下のような特徴を備えています。



オブジェクト指向プログラミング言語

OZシステムが提供するOZ言語は多重継承をサポートしたオブジェクト指向プログラミング言語です。

容易な型変換

キャストを行わなくても代入を行うことができる仕組みとして、強制代入演算子を導入しました。継承関係にあるクラスの代入では、その正確なクラス型を指定する必要がありません。


並行プログラミングサポート

分散環境上のオブジェクトは、複数のオブジェクトからの要求を並行して処理できる必要があります。そのための機構として、 OZシステムではモニタによる相互排除と条件変数による排他制御、同期機構を採用しています。その機構を利用するプログラミングを行うことができます。
