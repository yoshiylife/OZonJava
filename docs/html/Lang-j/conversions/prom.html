数値の昇格（Numeric Promotion）




数値型をオペランドにする単項演算式および二項演算式の中には、オペランドの型を、共通の型 に変換して処理されるものがあります。この時オペランドに対して行 う型変換を 「数値の昇格」と呼びます。

以下に、各演算式に関する「数値の昇格」規則を示します。


単項演算式

  • オペランドの型が byte型、short型 であれば、int型 に変換されます。
  • それ以外は、オペランドの型そのものが用いられます。




二項演算式

  1. 一方の型が double 型であれば、他方も double 型に変換されます。
  2. 1 でない場合、一方の型が float型であれば、他方も float型 に変換されます。
  3. 3.2 でない場合、一方の型が long 型であれば、他方も long 型に変換されます。
  4. 4.3 でない場合、両方の型を int 型に変換されます。
