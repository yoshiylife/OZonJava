型変換の種類


ここでは、OZ言語の型変換の種類について説明しています。



恒等変換 (Identity conversions)

同じ型への変換を恒等変換といいます。
任意の型は、それと同一の型へ変換することができます。


基本型のワイドニング変換 (Widening primitive conversions)

以下の基本型の変換を基本型のワイドニング変換といいます。

From To
byte short int long float double
short int long float double
int long float double
long float double
float double

たとえば、単純な代入演算子によって基本型のワイドニング変換が行われます。


基本型のナローイング変換 (Narrowing primitive conversions)

以下の基本型の変換を基本型のナローイング変換といいます。

From To
short byte
int byte short
long byte short int
float byte short int long
double byte short int long float

強制代入演算子によって基本型のナローイング変換が行われます。


クラス型のワイドニング変換 (Widening class conversions)

以下のクラス型の変換をクラス型のワイドニング変換といいます。

  • 子孫クラスSからその上位クラスTへの変換

  • null から任意のクラスへの変換


強制代入演算子によって子孫クラスSからその上位クラスTへの変換が行われます。


クラス型のナローイング変換 (Narrowing class conversions)

以下のクラス型の変換をクラス型のナローイング変換といいます。

  • 上位クラスTからその子孫クラスSへの変換

  • null から任意のクラスへの変換



クラス型への変換で、実際にリファレンスする値が型に適合しない場合には、システム例外ClassCastExceptionが発生します。
