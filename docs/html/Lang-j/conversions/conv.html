メソッド起動変換規則




メソッド本体の定義における戻り値型と return文のオペランドの型の関係

void でないメソッドの実装定義における戻り値型と return文のオペランドの型には、代入と同様の規定が適用されます。
すなわち、戻り値の型が C の場合、return 文のオペランドの型 D は C に同一であるか、あるいは、代入等価でなければいけません。＜恒等変換＞＜基本型のワイドニング変換＞＜クラス型のワイドニング変換＞

整数定数の特別規則も適用されます。



メソッド起動およびインスタンス生成の実引数代入

メソッド起動およびインスタンス生成の実引数代入には、代入の規則から整数定数の場合の特別な規則を除いたものが適用されます。＜恒等変換＞＜基本型のワイドニング変換＞＜クラス型のワイドニング変換＞
