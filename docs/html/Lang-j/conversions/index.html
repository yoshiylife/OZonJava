型変換（Type Conversions）


ここでは、OZ言語の型変換について説明しています。


OZ言語ではすべての式は型を持っています。演算子や文の構造が特定の型を要求します。その型と一致した型を持つものでなければ認められない場合もありますが、代入等価な型に解釈されて実行可能な場合もあります。演算子、メソッド起動とその戻り値、仮引数と実引数は型の等価性が要求されます。

OZ言語の型変換には5つの種類があります。

  • 恒等変換 (Identity conversions)
  • 基本型のワイドニング変換 (Widening primitive conversions)
  • 基本型のナローイング変換 (Narrowing primitive conversions)
  • クラス型のワイドニング変換 (Widening class conversions)
  • クラス型のナローイング変換 (Narrowing class conversions)


以下の2つの場面で型変換が行われます。

  • 代入変換
  • メソッド起動変換



OZ言語には、キャスト演算子はありませんが、それに代替するものとして強制代入演算子があります。強制代入演算子のために、基本型やクラス型の代入が容易になっています。
