型（Types）と変数(Variables)


OZ言語のすべての変数、すべての定数（リテラル）は１つの型を持っています。型によって、変数が保持できる値が決定されますし、また、実行できる操作も決定されます。

OZ言語の型には、基本型（Primitive Types）とクラス型（Class Types）の2種類があります。基本型とはOZ言語で規定される（predifined）型のことです。クラス型は新しい型を定義します。OZのオブジェクトはすべてクラス型のインスタンスです。

基本型とクラス型では、操作に対する値の取扱いが異なります。代入を例にとると、基本型の代入は値のコピーですが、クラス型の代入はリファレンスのコピーになります。


ソースプログラム中で、型を使用して以下のものが表現されます。

  • クラスのアトリビュート(Attributes）
  • メソッド引数（Method Parameters）
  • メソッドの戻り値（Method Results）
  • コンストラクタ引数（Constructor Parameters）
  • ローカル変数(Local Variables)
    ローカル変数宣言文だけでなく、for文の初期化文でもローカル変数が使用されます。
  • 例外ハンドル引数
  • クラス型のインスタンス生成

OZ言語ではさまざまな式が規定されていますが、すべての式が型を持っています。OZ言語の変数や定数、上にリストしたものはすべて式です。
