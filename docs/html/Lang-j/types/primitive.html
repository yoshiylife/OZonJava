基本型（Primitive Types）



OZ言語の基本型として以下の型が定義されています。

  • 数値型

    数値型には
    整数型
    浮動少数型
    があります。

  • 文字型

  • 論理型

  • condition型


それぞれの型には、とりうる値の範囲があります。OZ言語では変数もリテラルも型を持っています。すなわり、OZ言語のすべての式には型があります。



ここでは、基本型の種類とその型がとりうる値の範囲と型が持つ操作について説明します。




数値型 （Numeric Types）：整数型 (Integral Types)
整数を表す型には以下の4つの型があります。
byte short int long
整数型は符号付き整数を表現します。


型の値の範囲

Table.整数型の値の範囲
型 サイズ 最小値 最大値
byte 8ビット -128 127
short 16ビット -32768 32767
int 32ビット -2147483648 2147483647
long 64ビット -9223372036854775808 9223372036854775807



整数型がもつ操作（Integer Operations）

整数型は、以下の操作を持っています。

単項演算子（Unary Operators）

整数の演算を行う単項演算子。
単項演算子 +、 -
前置インクリメント ++
前置デクリメント --
後置式 ++、--

操作の戻り値の型は整数型。

算術演算子（Arithmetic Operators）
乗法演算子 *、/、 %
加法演算子 +、 -

操作の戻り値の型は整数型。

関係演算子（Relational Operators)

<、 <=、 >、 >=

返値の型は論理型。

等値演算子（Equality Operators)

==、 !=

返値の型は論理型。

条件演算子 （Conditional Operator）

? ：

操作の戻り値の型は整数型。(条件演算式の型の詳細については「条件演算子」を参照してください。)


Note: IEEE 754 の仕様に準拠の値と操作を持ちます。




例: 整数型 

     byte  b_infodata;        // byte型のローカル変数の宣言
     short request_short_num; // short型のローカル変数の宣言
     int   request_number;    // int型のローカル変数の宣言
     long  population_number; // long型のローカル変数の宣言







数値型 （Numeric Types）：浮動小数型 (Floating-Point Types)
浮動小数型には以下の2つの型があります。
float double
float型は32ビット単精度、double型は64ビット倍精度のIEEE754浮動小数を表現します。


型の値の範囲

float型は以下の図の値範囲が許されます。


dowble型は以下の図の値範囲が許されます。

Table.浮動小数型の値の範囲
型 サイズ 最小値 最大値
float 32ビット ±3.40282347E+38 ±1.40239846E-45
double 64ビット ±1.79769313486231570E+308 ±4.94065645841246544E-324


浮動少数型がもつ操作（Floating-Point Operations）

浮動少数型は、以下の操作を持っています。

単項演算子（Unary Operators）

浮動少数の演算を行う単項演算子。
単項演算子 +、 -
前置インクリメント ++
前置デクリメント --
後置式 ++、--

操作の戻り値の型は浮動少数型。

算術演算子（Arithmetic Operators）
乗法演算子 *、/、 %
加法演算子 +、 -

操作の戻り値の型は浮動少数型。

関係演算子（Relational Operators)

<、 <=、 >、 >=

返値の型は論理型。

等値演算子（Equality Operators)

==、 !=

返値の型は論理型。

条件演算子 （Conditional Operator）

? ：

操作の戻り値の型は浮動少数型。(条件演算式の型の詳細については「条件演算子」を参照してください。)


Note: IEEE 754 の仕様に準拠の値と操作を持ちます。




例: 浮動少数型 

    float  f;    // float型のローカル変数の宣言
    double d;    // double型のローカル変数の宣言







文字型 (Character Type)

文字型には以下の1つの型があります。

char
文字型は、Unicodeの文字コードを表現します。


型の値の範囲

この型の値は、 \u0000 から \uFFFF までの値 を持つことができます。
(\はバックスラッシュです。）


文字型がもつ操作（Character Operations）

文字型は、以下の操作を持っています。

等値演算子（Equality Operators)

==、 !=

返値の型は論理型。




例: 文字型 

    char sep = ozhome->startsWith ("/") ? '/' : '\\';
        // char型のローカル変数の宣言、初期値の設定








論理型 (Boolean Type)

論理型には以下の1つの型があります。

bool
論理型は論理値を表現します。


型の値の範囲

この型の値は、 true と falseです。


論理型がもつ操作（Boolean Operations）

論理型は、以下の操作を持っています。

単項演算子（Unary Operators）

論理型の演算を行う単項演算子。
単項演算子 !

操作の戻り値の型は論理型。

等値演算子（Equality Operators)

==、 !=

返値の型は論理型。

論理演算子（Logical Operators)

&&、 ||

返値の型は論理型。

条件演算子 （Conditional Operator）

? ：

操作の戻り値の型は論理型。(条件演算式の型の詳細については「条件演算子」を参照してください。)


論理型の式が用いられる文

  • if 文 (The if Statement)
  • while 文 (The while Statement)
  • do 文 (The do Statement)
  • for 文 (The for Statement)




例: 論理型 

   bool eof = false;  // 論理型のローカル変数の宣言、初期値の設定








condition型
condition 型には以下の1つの型があります。
condition
condition 型は並行実行される複数のスレッド間の同期をとるために利用する変数の型です。
すなわち、条件変数 を表現します。


この型の値の範囲、型が持つ操作は実行機構によって規定されます。


OZシステムでは実行機構としてOzVMを提供しています。
OzVMは、condition型の変数に対して、signal と wait という操作を提供しています。この詳細については「同期機構」を参照してください。



例: condition型 
  class RandezvouPoint { 
      condition randezvou;

      public locked void waitRandezvou () {
          wait (randezvou);
      }
      public locked void signalRandezvou () {
          signal (randezvou);
      }
   }
   // RandezvouPointクラスは 
   // condition型のアトリビュートを宣言し、
   // メソッド本体の定義でそれを利用しています。
