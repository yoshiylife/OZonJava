式（Expressions）


ここでは、OZ言語の式について説明しています。

式は評価されその結果として通常は変数、値を示します。これを単に式の値と呼びます。void という式は値を持たない唯一の例外です。
また、式が副作用を持つこともあります。副作用を持つ式は式文として用いられます。

OZ言語では、式も型を持っています。
式の値は式の型と代入等価です。また式の評価に際して型変換と型の昇格が発生します。これらについては「型変換と型の昇格」を参照してください。
