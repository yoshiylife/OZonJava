算術演算子（Arithmetic Operators）



ここでは、OZ言語の算術演算子について説明しています。


OZ言語には、以下の5つの算術演算子があります。

* / % + -


*、/、% を乗法演算子（Multiplicative Operators）、+ と - を加法演算子(Additive Operators)といいます。

等値演算子は左結合的（left-associative）です。


算術演算子を用いる式は二項演算式を形成します。



乗法演算子（Multiplicative Operators）

* 積を計算します
/ 商を計算します
% 剰余を計算します。



LALR(1)文法 
  binary_exp_0 : binary_exp_0 binary_operator_0 unary_exp
                 unary_exp
                    
  binary_operator_0 : *
                      /
                      %


説明

乗法演算子の2つのオペランドはともに数値型でなくてはいけません。
オペランド間で型の昇格が発生します。そして、式はその昇格された型を持ちます。
その型がfloat型、あるいは、double型であれば、浮動少数計算（floating-point arithmetic）が行われます。その他の数値型であれば整数計算（integer arithmetic）が行われます。

% は整数型にのみ有効です。




加法演算子

+ 和を計算します
- 差を計算します



LALR(1)文法 
  binary_exp_1 : binary_exp_1 binary_operator_1 binary_exp_0
                 binary_exp_0
                    
  binary_operator_1 : +
                      -


説明

加法演算子の2つのオペランドはともに数値型でなくてはいけません。
オペランド間で型の昇格が発生します。そして、式はその昇格された型を持ちます。
その型がfloat型、あるいは、double型であれば、浮動少数計算（floating-point arithmetic）が行われます。その他の数値型であれば整数計算（integer arithmetic）が行われます。
