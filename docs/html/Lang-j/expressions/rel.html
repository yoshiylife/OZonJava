関係演算子（Relational Operators)



ここでは、OZ言語の関係演算子について説明しています。


OZ言語には、以下の4つの関係演算子があります。

< > <= >=


関係演算子は左結合的（left-associative）です。すなわち、i<j<k は( i<j)<k)と解釈されます。


関係演算子を用いる式は二項演算式を形成し、式の値は論理型です。



関係演算子

< 左のオペランドの値が右のオペランドの値より小さいとき true、それ以外は false の値を持ちます。
> 左のオペランドの値が右のオペランドの値より大きいとき true、それ以外は false の値を持ちます。
<= 左のオペランドの値が右のオペランドの値より小さいか等しいとき true、それ以外は false の値を持ちます。
>= 左のオペランドの値が右のオペランドの値より大きいか等しいとき true、それ以外は falseの値を持ちます。



LALR(1)文法 
  binary_exp_2 : binary_exp_2 binary_operator_2 binary_exp_1
                 binary_exp_1
                    
  binary_operator_2 : <
                      >
                      <=
                      >=


説明

関係演算子の2つのオペランドはともに数値型でなくてはいけません。
オペランド間で型の昇格が発生します。そして、式はその昇格された型を持ちます。
その型がfloat型、あるいは、double型であれば、浮動少数比較 (floating-point comparison)が行われます。その他の数値型であれば符号付き整数比較(signed integer comparison)が行われます。

式の値は、論理型になります。




例: 関係演算子 
1:  while (true) {
2:      int size = is->available ();
3:      if (size >= 3) break;
4:  }
  • 3行目でint型の変数sizeが値3よりも大きければ、この関係演算子を利用した式の値は true になります。そうでなければ式の値はfalseです。
