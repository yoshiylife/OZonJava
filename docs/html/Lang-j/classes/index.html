クラス（Cｌasses）


OZシステムでは、クラス宣言で新しいクラス型を定義することができます。OZシステムのあらゆるオブジェクトはクラスとして宣言された型から生成されます。


OZ言語のクラスは以下の要素から構成されます。

アトリビュート
オブジェクトが持つ情報。メソッドによって操作されるデータ。
メソッド
オブジェクトに対して実行される手続き。
コンストラクタ
クラスからインスタンスを生成するための手続き。


これらを総称して、クラスのメンバと呼びます。

通常のクラスは上位クラスから継承して宣言されます。上位クラスの存在しないクラスをシステム定義のクラスとして提供しています。

OZシステムでは実効力のあるバージョン管理を行うためにクラス宣言を2つのパート、すなわち、インタフェース部と動作部を分けて管理するという仕組みを導入しています。そして、インタフェース部の継承と動作部の継承を別々に定義することができます。
