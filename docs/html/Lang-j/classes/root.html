Rootクラス（Root Class）



OZシステムのclass型クラスは、すべてRootクラスの子孫クラスです。Rootクラスは、OZクラスライブラリに格納されており、クラスの基本的な振舞いを定義しています。


OZシステムのクラスには以下のような特徴のある機能を持っています。

排他制御機能
OZシステムでは、オブジェクト単位の排他制御を行います。
セキュリティ機能
OZシステムのオブジェクトは、セキュリティからみて自分が安全かどうかを判断する機能を持っています。



排他制御については、「同期機構」を参照してください。セキュリティ機能については「プログラミングガイド」の該当個所を参照してください。


ここでは、上記の特徴以外のRootクラスのメソッドを概略します。
なお、Rootクラスのコンストラクタはプログラムから起動することができません。

メソッド 意 味
public final boolean equals(Root arg1) arg1のオブジェクトと当オブジェクトが一致するかを調べます。
public final int hashCode() オブジェクトのハッシュ値を答えます。
下位クラスで再定義できません。
public String asString() オブジェクトの文字列表現を答えます。
public final String getGOD() 当オブジェクトが属するセルのGODを答えます。
public final String getGOL() 当オブジェクトが属するセルのGOLを答えます。
public final boolean isRed() 中断されているかどうかを返します。


Threadクラスのオブジェクトを生成するときには、Runnableクラスのオブジェクトがコンストラクタ引数として渡されます。RunnableクラスはRunメソッドを持っており、Threadクラスのオブジェクトが生成されると同時にそのRunメソッドが起動されます。


Threadクラスの詳細は、「クラスライブラリ」の該当する個所を参照してください。
