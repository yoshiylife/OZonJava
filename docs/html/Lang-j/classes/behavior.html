オブジェクトの動作


OZシステムにおけるオブジェクトはクラス型のインスタンスです。class型のインスタンスもとcell型のインスタンスもオブジェクトです。
ここでは、オブジェクトの動作について説明しています。



メソッド起動式

オブジェクトはメソッド起動によって動作します。

メソッドを起動させるオブジェクトを特定して、そのメソッドを起動します
オブジェクトへのリファレンスを保持する式に対して、実行するメソッドを指定することで、オブジェクトを動作させることができます。

メソッド起動のための式をメソッド起動式と呼びます。

メソッド起動式:[ クラス型の式 −＞ ] メソッド名( [ 引数リスト ] )



メソッドの型

メソッドは型を持っています。メソッド起動式の戻り値は、そのメソッド型と適合する型を持っています。



例: メソッド起動式 

 1: public void addContent(FMObject cont) {
 2:     Overseer over=>create();
 3:     over->permit(cont);
 4:     addContentInternal(cont);
 5: }
  • 3行目で、Overseerクラスのインスタンスoverの持つメソッドを起動しています。




例: メソッド起動式 

 class Garage {
     KeyGenerator     keyGen;
     Dictionary       valueTable;

     new create()  {
         KeyComparator  keyComp;
         keyGen => create();
         keyComp => create();
         valueTable => create(keyComp);
     }
     public Root get(Key key){
         return valueTable->get(key);
     }

     public Collection keys() {
         return valueTable->keys();
     }
     ...
 }
  • メソッドget(Key key)とkeys()のメソッド本体の定義で、アトリビュートに対するメソッド起動式が記述されています。
