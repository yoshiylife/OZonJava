名前空間（Names）


ここでは、OZ言語の名前空間について説明しています。オブジェクト、メンバ、型、ラベルなどは名前を持っています。名前は識別子で表現されます。そして、それが有効となるスコープがあります。

OZ言語のクラスは、１つのアプリケーション内でクラス名の重複がなければ、同一名のクラスを定義することができます。クラスの一意性を表現するものがクラス識別子だからです。したがって、別々に開発されたクラスライブラリを同時に利用してアプリケーションを構築することができます。


識別子と名前


OZ言語では、以下の言語要素が名前を持ちます。

  • クラス
  • アトリビュート
  • メソッド
  • コンストラクタ
  • メソッド仮引数（メソッド定義で宣言されます。）
  • コンストラクタ仮引数
  • try 文の catch 節で宣言される例外の仮引数
  • ローカル変数
  • for 文の初期化文で宣言されるローカル変数
  • ラベル


名前は識別子で表現されます。
