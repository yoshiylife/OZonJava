同期機構


OZ言語はマルチスレッドプログラミング言語です。OZシステムが提供する実行環境OzVMは同時に複数のスレッドを実行することができ、排他制御と同期の機構を提供しています。排他制御のためにはモニタを採用しています。そして、条件変数によって同期をとることができます。

OZ言語のcondition型は条件変数を表現する型であり、その値と操作は実行環境に依存します。OZシステムの実行環境は条件変数に対して、wait()とsignal()という操作を提供します。
