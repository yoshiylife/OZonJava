スレッド（Thread）



OZシステムのスレッドは、Threadクラスのインスタンスです。

Threadクラスは、OZクラスライブラリに格納されています。

上位実装クラスとして、Rootクラスを持ち、以下のメソッドメンバを持っています。

メソッド 意 味
new create(Runnable arg1) コンストラクタ。
新しいスレッドを生成します。
引数にはRunnableクラスのオブジェクトを指定します。
public void join(long arg1) スレッドの実行終了を待ち合わせます。
引数にタイムアウトの時間を設定します。
public void join() スレッドの実行終了を待ち合わせます。
public boolean isAlive() スレッドがまだ実行中かどうかを調べます。
public void interrupt() スレッドの実行を中断させます。
public boolean isInterrupted() 中断されているかどうかを返します。


Threadクラスのオブジェクトを生成するときには、Runnableクラスのオブジェクトがコンストラクタ引数として渡されます。RunnableクラスはRunメソッドを持っており、Threadクラスのオブジェクトが生成されると同時にそのRunメソッドが起動されます。


Threadクラスの詳細は、「クラスライブラリ」の該当する個所を参照してください。
