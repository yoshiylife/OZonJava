識別子（Identifiers）


英字 (a-zA-Z) から始まり英数字 (a-zA-Z0-9) およびアンダースコア (_) からなる任意の長さの文字列で、以下の字句を含まないものが識別子です。

  • キーワード
  • 空白文字


OZ言語では、英字の大文字 (A-Z) と小文字 (a-z) を区別します。

識別子は変数名の宣言､変数の参照､ラベル名などで使用されます。詳細は「名前」を参照してください。



例: 識別子 

     1:/**
     2: * FMObjectは、Facility Manageidentifiernt Systemでの管理対象である
     3: *「リソース」を表すオブジェクトである。
     4: */
     5: 
     6:class FMObject : *Root {
     7:        protected Pointer identifier;
     8:        String name;
     9:        Set relations;
    10:        RelationComparator rc;
    11:
    12:        //FMObjectの生成
    13:        new create() {
    14:                rc=>create();
    15:                relations => create(rc);
    16:                name => breed("UnKnown"); //UnKnownは初期値
    17:        }
      :      ....
      : }


  • 6行目の FMObject,、Root は識別子です。
  • 7行目の Pointer、identifier は識別子です。
  • 8行目の String、name は識別子です。
  • 9行目の Set、relations は識別子です。
  • 10行目の RelationComparetor、rc は識別子です。
  • 13行目の create は識別子です。
  • 14行目の rc、create は識別子です。
  • 15行目の relation、create、rc は識別子です。
  • 16行目の name、breed は識別子です。
