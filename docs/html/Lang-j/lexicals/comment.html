コメント (Comments)


次の2種類の形式でコメントが記述できます。


/* から */ までの文字列
この形式のコメントの入れ子はできません。
このコメント中のコメント文字列（// と /*）は特別な意味を持ちません。
// から改行文字 までの文字列
このコメント中のコメント文字列（/* と */）は特別な意味を持ちません。



コメントは字句の区切りとしてのみ扱われます。
改行文字については「空白文字」を参照してください。



例: コメント 

     1:/**
     2: * FMObjectは、Facility Manageidentifiernt Systemでの管理対象である
     3: *「リソース」を表すオブジェクトである。
     4: */
     5: 
     6:class FMObject : *Root {
     7:        protected Pointer identifier;
     8:        String name;
     9:        Set relations;
    10:        RelationComparator rc;
    11:
    12:        //FMObjectの生成
    13:        new create() {
    14:                rc=>create();
    15:                relations => create(rc);
    16:                name => breed("UnKnown"); //UnKnownは初期値
    17:        }
      :      ....
      : }


  • 1-4行目と12行目はコメント。
  • 16行目の //以降の文字列(UnKnownは初期値)はコメント。
