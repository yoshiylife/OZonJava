字句 （Lexical Structure）


OZ言語では、以下の字句を規定しています。

  • コメント
  • 識別子
  • キーワード
  • リテラル
  • 演算子
  • 区切子
  • 空白文字


識別子、キーワード、リテラル、演算子、区切子はトークン(Token)と呼ばれます。識別子、キーワード、リテラルを識別するためには、演算子、区切子、あるいは、空白文字が必要です。

OZコンパイラがソースプログラムに定義されたトークンを翻訳して、実装言語である Java 言語の構文要素に変換します。






  • コメント （Comments）

  • 識別子(Identifiers)

  • キーワード (Keywords)

  • リテラル (Literals)

  • 演算子(Operators)

  • 区切子(Separators)

  • 空白文字 (White space)
