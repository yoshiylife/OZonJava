do 文（The do Statement）


式の値によって、文の実行を制御します。

構文

do {文} while (式);

式は論理型でなくてはいけません。

LALR(1)文法 

  do_stmt : DO stmt WHILE ( exp ) ;



説明

式の評価を行う前に文の実行が行われます。文の実行が正常に行われると、式の評価が行われます。そして、その値が true ならば、再度文を実行します。式の評価の値がfalseになるまで繰り返します。

break文によって、文の実行を中断し、do文を終了することができます。
ラベルのないcontinue文によって、文の実行を中断し、式の評価に処理を進めることができます。
ラベル付きのbreak文があると、そのラベルまで処理が進みます。
ラベル付きのcontinue文があると、そのラベルまで処理が進みます。



例: do 文 

   do {
      ;                  // 空文
    } while (j++ < i);   // iとjはint型です。jがiよりも小さい間ループします。
