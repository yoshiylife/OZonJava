continue 文（The continue Statement）


状況によって文の実行を制御できます。

構文

continue [ラベル名] ;

ラベル名を省略することができます。ラベル名を省略したcontinue文をラベルなしcontinue文と呼びます。
ラベル名が省略されていないcontinue文をラベル付きcontinue文と呼びます。


LALR(1)文法 

  jump_stmt : CONTINUE identifier_or_empty ; 

  identifier_or_empty : /* empty */
                        IDENTIFIER



説明

continue文は、while文,、do文、for文の内側で定義できます。すなわち、繰返しを行う文の中で定義されます。

ラベルなしcontinue文は、そこで文の処理を終了し、次の繰返しの開始に処理が移されます。最も内側のswitch文,、while文,、do文、for文の正常に終了させます。識別子を持たないbreak文は while文,、do文、for文の内側に定義しなくてはいけません。

ラベル付きcontinue文は、そのラベル名のラベル文に処理を移しますが、このラベルは繰返し単位の内側に宣言されていなくてはいけません。



例: continue文 

1:  for (int i = 0; i < 10; i++) {
2:      if (i % 2 == 0) 
3:          continue;
4:      int k = i;
5:      for (int j = 0; j < k * 10; j++)
6:          ;
7:  }

3行目のcontinue文が実行されると、文の実行が終了し式（"i < 10"）が評価されます。
