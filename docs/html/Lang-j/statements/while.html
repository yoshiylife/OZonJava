while 文（The while Statement）


式の値が false になるまで、文の実行を繰り返します。

構文

while (式) 文

式は論理型でなくてはいけません。




LALR(1)文法 

  while_stmt : WHILE ( exp ) stmt
                    
  no_short_if_while_stmt : WHILE ( exp ) no_short_if_stmt


説明
まず、式の評価が行われます。そして、その値が true ならば文を実行します。すべての文の実行が正常に行われると、再度式の評価が行われます。式の評価の値がfalseになるまで繰り返します。

break文によって、文の実行を中断し、while文を終了することができます。
ラベルのないcontinue文によって、文の実行を中断し、式の評価に処理を進めることができます。
ラベル付きのbreak文があると、そのラベルまで処理が進みます。
ラベル付きのcontinue文があると、そのラベルまで処理が進みます。



例: while文 

  while (it->hasMoreElements ()) {
      String src;
      src =| it->nextElement ();
      compiler_server->compile (src, school, log);

      File f=>breed (src);
      f->delete ();
  }



例: while文 break文で繰返しを終了する 

  while (true) {
      String line = in->readLine ();
      if (line == null)
        break;

      out->write (line);
      out->write ("\n");
  }
