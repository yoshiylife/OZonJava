ローカル変数宣言文（Local Variable Declaration Statements）


ローカル変数宣言文はローカル変数を宣言する文です。
ローカル変数は型を持っていますから、型名とローカル変数名を対にして宣言します。


構文
基本型のローカル変数の宣言は以下のように行います。

型名 ローカル変数名［ = 初期値 ］;

初期値は省略できます。省略された場合にはデフォルトの値が設定されます。デフォルトの値は型によって決定されます。「変数」を参照してください。


型がクラス型の場合のローカル変数の宣言は以下のように行います。

型名 ローカル変数名 ［ => コンストラクタ呼出し］;

コンストラクタ呼出しでインスタンス生成が指定されます。
コンストラクタ呼出しは省略できます。省略された場合にはデフォルトの値nullが設定されます。


LALR(1)文法 （以下の\ は次の行への継続を示しています） 

  declaration_stmt : type_name identifier_only_or_with_initializer_list;

  identifier_only_or_with_initializer_list : \
                identifier_only_or_with_initializer_list , \
                          identifier_only_or_with_initializer
                identifier_only_or_with_initializer
                                            
  identifier_only_or_with_initializer : IDENTIFIER
                                        identifier_with_initializer
                                    
  identifier_with_initializer : => instantiate_exp_for_local_variable
                                IDENTIFIER = exp


説明
ローカル変数宣言文はブロック内でのみ記述できます。
ローカル変数名のスコープはそのブロック内です。外側のブロックですでに宣言されている 識別子と同じ名前のローカル変数を宣言した場合、その宣言以降そのブロック内では外側のブロックの識別子が隠されます。その結果、そのブロックでは新たに宣言した識別子（ローカル変数名）が有効となり、そのブロックが終了すると再び外側 で宣言していた識別子が有効となります。「名前とスコープ」を参照してください。



例: ローカル変数宣言文 

  int request_number; // int型の宣言文
  String command;     // Stringクラスの宣言文
  StringComparator scomp=>create ();
  // StringComparator型の宣言、コンストラクタ呼出しを行う
