文（Statements）


ここでは、OZ言語の文について説明しています。

文はOZプログラムの実行を制御するものです。そして、値を持ちません。指示された場合以外は順次実行されます。

文が別の文を構成要素として持つこともできます。


文の種類
OZ言語が規定する文Oとしては以下のものがあります。LALR(1)文法で記述しています。LALR(1)文法による構文記述は「LALR(1)文法」にまとめてあります。

  stmt : no_subsequent_stmt
         label_stmt
         if_then_stmt
         if_then_else_stmt
         while_stmt
         for_stmt

  no_subsequent_stmt : empty_statement
                       exp_stmt
                       compound_stmt
                       switch_stmt
                       do_stmt
                       jump_stmt
                       throw_stmt
                       try_stmt

  no_short_if_stmt : no_subsequent_stmt
                     no_short_if_label_stmt
                     no_short_if_if_then_else_stmt
                     no_short_if_while_stmt
                     no_short_if_for_stmt
