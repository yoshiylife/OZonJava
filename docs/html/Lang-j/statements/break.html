break 文（The break Statement）


状況によって文の実行を制御できます。

構文

break [ラベル名] ;

ラベル名を省略することができます。ラベル名を省略したbreak文をラベルなしbreak文と呼びます。ラベル名が省略されていないbreak文をラベル付きbreak文と呼びます。


LALR(1)文法 

  jump_stmt : BREAK identifier_or_empty ; 

  identifier_or_empty : /* empty */
                        IDENTIFIER



説明

ラベルなしbreak文は、最も内側のswitch文,、while文,、do文、for文の正常に終了させます。このbreak文は switch文,、while文,、do文、for文の内側に定義しなくてはいけません。

ラベル付きbreak文は、ラベル文の中の任意の場所で記述できます。このbreak 文の実行後、そのラベルを指定したラベル文の次の文に制御が移動します。


例: 繰返し文に記述されたラベルなしbreak文 

    Iterator iter = pool->iterator();
    FMObject obj;
    Pointer point;

    while (iter->hasMoreElements()) {
        point =| iter->nextElement();
        obj = point->stuff();
        if (obj->getName()->isequal(name)) {
            pool->remove(point);
            break;        // while文の繰返しを終了する
        }
    }



例: ラベル付きbreak文 

 1:  for_loop:
 2:    for (int i = 0; i < a; i++)
 3:      if (i % 2 == 0) 
 4:          continue for_loop;
 5:      else 
 6:          switch (i % 5) {
 7:          case 0:
 8:              break;
 9:          case 1:
10:              j++;
11:              break;
12:          case 2:
13:              j += 2;
14:              break;
15:          default:
16:              break for_loop;
17:          }
18:      while (j > a)
19:           j--;
20:      do {
21:           ;
22:      } while (j++ < i);
23:      return j + i;
24:  }
  • 8行目と11行目のラベルなしbreak文が実行されると、18行目に制御が移ります。
  • 16行目のラベル付きbreak文が実行されると、1行目に制御が移ります。
