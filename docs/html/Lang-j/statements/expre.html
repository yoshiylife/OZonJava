式文（Expression Statements）


式を評価する文です。

構文

式 ;

セミコロンで終了します。


LALR(1)文法 

  exp_stmt : exp_of_exp_stmt ;
                    
  exp_of_exp_stmt : assign_exp
                    pre_increment_or_decrement_exp
                    post_increment_or_decrement_exp
                    method_invoke_exp
                    instantiate_exp



説明

式文中で使用できるのは以下の式です。

  • 代入式
  • 前置インクリメント式
  • 後置インクリメント式
  • メソッド起動式
  • インスタンス生成式

すなわち、副作用を持つ式だけが使用できます。
上記以外の式は副作用を持ちません。 単独では意味を持たないため、式文中では使用できません。



例: 式文 

  birthDay = obirthDay;   // 式文...代入式
  counter++;                // 式文...後置インクリメント式
  super->trace(mark);     // 式分...メソッド起動式
