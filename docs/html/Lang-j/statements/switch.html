switch 文（The switch Statement）


式の値によって、文の実行を制御します。

構文

switch (式) {文}

式はint型でなくてはいけません。


LALR(1)文法 

  switch_stmt : SWITCH ( exp ) switch_block
                    
  switch_block : { switch_block_stmts switch_labels_or_empty }
                    
  switch_block_stmts : /* empty */ 
                       switch_block_stmts switch_block_stmt
                    
  switch_block_stmt : switch_labels stmt
                      stmt
                    
  switch_labels_or_empty : /* empty */
                           switch_labels
                       
  switch_labels : switch_label
                  switch_labels switch_label
                    
  switch_label : CASE exp :
                 DEFAULT :



説明

まず、式の評価が行われます。そして、その値が true ならば文を実行します。すべての文の実行が正常に行われると、再度式の評価が行われます。
文に、case 定数式: をつけることができます。この場合には、式の値と等しい定数式を持つ case のつけられた文から実行を開始します。
文に、default: をつけることができます。この場合には、式の値と等しい定数式を持つcaseが存在しない場合にdefault:のつけられた文から実行を開始します。1つのswitch文に、default:はたかだか１つだけしか含むことができません。


例: switch 文 

  public void processWindowEvent(WindowEvent event) {  
      switch(event->getID()) {
        case WindowEvent->getWINDOW_CLOSING():
            this->dispose();                    // Free the system resource
            this->setVisible(false);            // hide the Frame
      }
  }

// オブジェクトeventの持つ振舞いgetID()の返り値は整数型。
// WindowEventクラスの持つ振舞いgetWINDOW_CLOSING()の返り値も整数型です。
