throw 文（The throw Statement）


throw文によって例外を明示的に発生させることができます。
throw文で発生させた例外を、try文によって処理することができます。try文については「try文」を参照してください。例外処理については「例外処理」に説明しています。

構文

throw [式];

式は、クラス型の変数、あるいはクラス型の値でなくてはいけません。

try文のcatch節では、式を省略することができます。省略された場合には、catch節で捕獲された例外が
発生します。



LALR(1)文法 

  jump_stmt : RETURN exp_or_empty ;

  exp_or_empty : /* empty */
                 exp



説明

式の評価が行われて、その後例外が発生します。

例外として任意のオブジェクトを投げる（throw）ことができます。
式で表現されるのが例外で投げられる例外です。


例: throw文 

1: public Root invoke(MethodRunnable mr, int time){
2:     isDone = false;
3:     Thread t => create(mr);
4:     endWait(time);
5:     if (isDone) return(mr->getResult());
6:     else { TimeOutException e =>init(); throw e; }
7: }
  • 5行目のif文で判定結果がfalseの場合、TimeOutExceptionクラスのインスタンスが作成され、そして、例外が明示的に発生されます。例外として、TimeOutExceptionクラスのインスタンスが投げられます。
