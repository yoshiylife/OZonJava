ラベル文（Labeled Statement）


文にラベルをつけることができます。

構文

ラベル名 : 文



LALR(1)文法

  label_stmt : IDENTIFIER : stmt     
                    
  no_short_if_label_stmt : IDENTIFIER : no_short_if_stmt



説明

ラベルは、ラベル付き break文または continue 文の対象となります。


ラベル名のスコープ

ラベル名の名前空間は他の名前空間とは独立しています。したがって、同じラベル名が存在しない限りエラーにはなりません。クラス名、ローカル変数名などと同一の名前をラベル名として指定してもかまいません。
