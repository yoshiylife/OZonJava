複文（Blocks）


複文はブレースで囲まれた一連の文です。

構文

{ 文 }

文として宣言文が含まれていてもかまいません。

文としてブロック（｛文｝）が含まれていてもかまいません。

1つも文を含まないブロックも正常なブロックです。


LALR(1)文法 

  compound_stmt : { stmt_list }

  stmt_list : /* empty */ 
              stmt_list block_stmt

  block_stmt : stmt
               declaration_stmt


説明
ブロック内の文は順次実行されます。



例: 複文 セル宣言は複文です。 

 cell FedRoot { 
     protected System system;
     protected Authenticator auth;
     protected boolean debug; 
     ....
 }

例: 複文メソッド宣言は複文です。 

 public void authorize(Root obj) {
     turnGreen(obj);
     obj->toGreen();
 }
