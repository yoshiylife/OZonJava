if 文（The if Statements）


状況によって文の実行を制御します。

構文

if (式) 文

式は論理型でなくてはいけません。

if (式) 文1 else 文２

式は論理型でなくてはいけません。

文2には、すべての種類の文を指定できます。
文１には、上記形式のif文を含ませることができません。
また、上記形式のif文を含むwhile文を含ませることができません。参考： while (式) 文



LALR(1)文法 

  if_then_stmt : IF ( exp ) stmt
                    
  if_then_else_stmt : IF ( exp ) no_short_if_stmt ELSE stmt
                    
  no_short_if_if_then_else_stmt : IF ( exp ) no_short_if_stmt ELSE \
                                    no_short_if_stmt



説明

短い形式のif文では、式の値が true のとき、文が実行されて、if文が終了します。そして、式の値がfalseのときには、if 文が終了します。

else節を伴うif文では、式の値が true のとき、文1が実行されて、if文が終了します。
式の値がfalseのときには、文2が実行されて、if 文が終了します。



例: ｉｆ文 短い形式のif文 

      if (pbirthDay != null) {
          birthDay=>breed(pbirthDay->getTime());
      }


例: ｉｆ文if文の入れ子 

      if (birthDay == null) {
          if (obirthDay != null) return false;
      } else if (!birthDay->isEqualDate(obirthDay)) {
          return false;
      }
