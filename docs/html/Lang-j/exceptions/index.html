例外処理（Exceptions）




OZ言語では例外をハンドルすることができます。OZ言語の例外とはオブジェクトです。任意のオブジェクトを例外として扱うことができます。

OZシステムはシステム定義の例外をクラスライブラリとして提供しています。またユーザが例外を定義することもできます。


例外が発生する原因としては以下の場合が存在します。

  • OzVMが異常な実行を検知した場合。
    整数をゼロで割るというように、構文的には正しいにも関わらず異常な式の評価が行われた場合。
    リソースの制約を超えるような使用要求が行われた場合。

  • throw文が実行された場合。

  • 同期機構が正しく使用されなかった場合。


例外を発生させるためには、throw文を用います。
例外の捕獲してハンドルするためには、try文を記述します。
