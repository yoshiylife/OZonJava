I/O interface

本節では入出力ライブラリのインタフェースを示す。 OZでは、入出力ライブラリとして以下のものを提供する。 バイト単位での入出力を行うInputStream/OutputStreamとその派生クラス。 また文字単位での入出力を行うReader/Writerとその派生クラス これらを合わせて入出力ストリームとした。 またファイル操作、ソケット通信、外部プロセス起動、字句解析を 関連するクラスとして用意した。

  • 入出力ストリーム
  • ファイル
  • 外部プロセス
  • 字句解析
