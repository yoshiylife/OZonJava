タイマ API

  • abstract class MethodRunnable :* Runnable
    • new create(Timer t);
      MethodRunnable を生成する。これはユーザが作る、MethodRunnable を継承 した Runnable クラスのコンストラクタの中のみから呼ばれるので、ユーザ が直接使うことはない。
    • public Root getResult();
      タイマをかけて実行したメソッドの返り値を返す。 このメソッドは Timer の中から呼ばれるものであり、ユーザが直接使うこ とはない。
  • class Timer
    • new create();
      タイマを生成する。
    • public Root invoke(MethodRunnable mr, int time);
      ユーザが作る、MethodRunnable を継承した Runnable クラスのオブジェク ト mr とタイムアウト時間 time をもとに、タイマをかけてメソッド実行を 行なう。タイマをかけて実行したいメソッドの返り値が invoke の返り値と なる。タイマをかけて実行したいメソッドが発生させる例外はこの invoke の例外としてあがる。
  • class TimeOutException
    • new init();
      タイムアウトを示す例外であるTimeOutExceptionを生成する。 この例外は Timer の中から投げられるものであり、ユーザが直接生成するこ とはない。
