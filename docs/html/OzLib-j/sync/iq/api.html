同期構造体 API マニュアル

  • class IStruct
    • new create();
      IStructを生成する。
    • public locked void write(Root val);
      val の書き込みを行う。
      例外 IStructWriteException:二回以上の連続した書き込み
    • public locked Root read();
      読み込みを行う。
      例外 IStructReadException:二回以上の連続した読み込み
  • class QStruct
    • new create();
      QStructを生成する。
    • public locked void write(Root val);
      val の書き込みを行う。
    • public locked Root read();
      読み込みを行う。
