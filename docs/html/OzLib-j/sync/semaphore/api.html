セマフォ API

  • class BinarySemaphore
    • new create();
      排他モードのみのバイナリセマフォを作成する。
    • public locked void P();
      P 命令に相当するメソッドである。
    • public locked void V();
      V 命令に相当するメソッドである。
  • class ModedBinarySemaphore :* BinarySemaphore
    • new create();
      共有/排他モードのセマフォを作成する。
    • public locked void exclusiveP();
      クリティカルセクションに排他モードで進入する場合の P 命令に相当するメ ソッドである。
    • public locked void sharedP();
      クリティカルセクションに共有モードで進入する場合の P 命令に相当するメ ソッドである。
