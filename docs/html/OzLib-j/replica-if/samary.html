概要

複製ライブラリは指定されたオブジェクトを複製として管理するためのセルで、 プライマリコピー方式とピアコピー方式の二種類のセルが用意されている。 複製ライブラリは複製の正否による副作用を取り除くためにトランザクション ライブラリを用いている。
開発者はサブセルの UserCopyOfPrimaryCopy または UserCopyOfPeerCopy を開発 することにより、作成したオブジェクトを複製として処理することができる。
