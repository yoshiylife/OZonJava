リソースの予約

リソースの予約を行うには、予約の開始時刻と 終了時刻を設定したResereveオブジェクトを 用意し、予約可能なリソースの役割である Schedulerオブジェクトのreserve(Reserve) メソッドを利用する。
Schedulerは受け取ったReserveをもとに Schedulableに対しreserve(Reserve)を行う。 Schedulableは受け取ったReserveの予約開始 時刻と終了時刻内で予約可能ならば予約を行う。 予約成功後は、予約に用いたReserveオブジェクトを Schedulerと予約したオブジェクト間にアタッチする。
