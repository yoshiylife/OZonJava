FMObjectの管理主体間の移動

FMObjectが管理主体間を移動する時は FMContainerのmove(Pointer, Pointer)を用いて 行われる。このとき移動するFMObjectの 移動前と移動後のPointerをArchiverに登録する。
これにより、Archiver内に移動の履歴が残る。 この履歴は移動したFMObjectの追跡にも用いられる。
