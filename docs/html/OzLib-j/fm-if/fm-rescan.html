予約のキャンセル

予約のキャンセルは予約者とリソース(Schedulerオブジェクト)の 間にアタッチされた特定のReserveオブジェクトに対し、 cancel()メソッドを実行することで、実現される。 Reserveはcancel()を要求されると、自信のアタッチされている Schedulerに対し、cancel(Reserve)を要求する。 SchedulerはSchedulableに対しcancel(Reserve)を要求し、 タイムテーブル内のReserveにより指定された時間の予約の キャンセルを行う。
