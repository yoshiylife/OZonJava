リソースの探索

FMObjectは関連を持つ他のFMObjectとRelationに よって結ばれている(リソースのアタッチ参照)。 関連先のFMObjectは以下の手順により取得される。

  1. Stringによって表現された関連名により findRelation(String)を実行し対象となる Relationオブジェクトのリストを取得する
  2. 取得したリストからRelationオブジェクトを取り出す
  3. Relationオブジェクトに対しdestination()を実行し関連先オブジェクトのPointerを取得する
  4. 取得したPointerに対しstuff()を実行することで関連先のFMObjectのコピーを得る
  5. destination()

以上の手順によりFMObjectのチェインを辿ることにより、 メンテナンス対象のFMObjectの探索を行う。
