リソース内部情報の取得と変更

リソースであるFMObjectからの内部情報の 取得はgetName()を用いて行う。 FMObjectを継承し属性を追加した場合には その属性へのアクセスメソッドを用意しなければならない。
リソース内部の情報の変更を行うには、 setName(String)メソッドを用いる。 FMObjectを継承し属性を追加した場合には その属性の変更を行うメソッドを用意しなければならない。 また、変更しただけでは元リソースに反映 されないのため、commit()メソッドを用いて リソースに対する変更を元リソースに反映している。
