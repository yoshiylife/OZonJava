ファシリティ管理

ファシリティ管理は、利用者の定義する情報をリソースとして管理することを通し、 当該管理機構の利用者相互間での情報の共有を促すことを目的とした機構であ る。
広範なアプリケーションにおいて作成される情報に対しても適用可能とするた めに、ファシリティ管理はフレームワーク形式として構成されており、従って テンプレートとして提供されている箇所に対し、必要とされるリソース をフックとして利用者が定義してゆくことにより、ファシリティ管理の提供 するサービスの全体的な枠組を逸脱すること無く、しかも利用者固有の拡 張機能を付加して行くことが可能となっている。
またそのような個別の手動的な拡張を行わずとも、ファシリティ管理が本来提供する基 本リソース群を元にした利用者情報の蓄積と、それらリソース相互 間に有機的な関係を構築していくことにより、多面的な情報の共有と利用を支 援する環境として用いることもできる。
