ファシリティ管理 API

ファシリティ管理APIでは、ファシリティ管理機構内で 管理される情報単位である``リソース''に対し、基本機能として 以下の機能を提供する。

  • リソースの生成
  • リソースの削除
  • リソースの移動
  • リソースの照会
  • リソースのインポート
  • リソースのエクスポート
  • リソースのアタッチ
  • リソースのデタッチ
  • リソース公開の活性化・非活性化

また応用的な機能としてメンテナンス機能、予約、 スケジューリング、リソースの時間的管理、組織管理等の機能も 有する。以下では上に列挙したファシリティ管理機構の基本部分 について説明し、応用的な機能については後項で述べる。
