組織に配置されている人の呼び出し

組織に配置されている人の呼び出しは getMembers(),getMembers(String)により行う。 getMembers()はその組織に配置されている 人の全てを返し、getMembers(String)は Stringで示された名前を持つ人を組織内で探し、 これを返す。
具体的には、getMembers()はOrganization内の Poolの持つ全ての要素の集合を返す。 getMembers(String)は、Organizationの持つPool内の Actorに対しStringで指定された名前を持つものの 集合を検索結果として返す。
