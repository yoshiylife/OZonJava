予約の照会

予約の照会を行うには、照会したい時間の 開始時刻と終了時刻を設定したReserveを 用意し、Schedulerオブジェクトのconfirm(Reserve) メソッドを起動する。Schedulerは受け取った Reserveを用いて、Schedulableのconfirm(Reserve) を起動する。 Schedulableは指定された時間帯のタイムテーブルの 状況を予約状況として返す。 また、あるリソースのすべての予約状況は Schedulerに対しすべてのgetAllRelations()メソッドを用いて、 すべてのReserveを返すよう要求すればよい。
