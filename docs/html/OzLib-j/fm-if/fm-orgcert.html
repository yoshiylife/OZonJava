配置されている人の確認

配置されている人の確認はisBelong(Actor)、 isBelong(String)により行う。 前者はある人格を表すオブジェクトがこの組織に 含まれているかを確認する。 後者はある名前を持つ人格がこの組織に含まれているかを 確認する。
具体的には、isBelong(Actor)を実行すると、 Actorと同じActorがOrganization内のPoolに含まれていた場合 trueとなる。isBelong(String)を実行すると、Stringで指定された 文字列と同じ名前を持つActorがPoolに含まれていた場合trueとなる。 それ以外の場合は全てfalseが返る。
