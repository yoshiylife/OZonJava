FMObjectの生成

FMObjectの生成は管理主体(FMContainer)に登録される ことで行われる。FMContainerへの登録は add(FMObject)メソッドにより行われる。 この時、FMContainer内では渡されたFMObjectの管理と 時間的管理を行うKarteを生成し、これにFMObjectを登録する。 このときKarteにはFMObjectの生成日時が記録される。
