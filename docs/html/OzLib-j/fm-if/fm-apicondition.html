制約事項

  • FMContainerに登録されていないリソースは使用不可能
  • FMObjectを継承し、属性が追加された場合は update(FMObject), cloneSelf()の書き換え、アクセスメソッドの追加が必要
  • FMObjectを継承した場合、isEqualFMObject(FMObject)の振舞が最初に用意されているもので問題がある時はこれを書き換える必要がある
