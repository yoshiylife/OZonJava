機能詳細

ファシリティ管理機構のメンテナンス機能に就いて述べる。
ファシリティ管理機構において、その情報資源であるリソースは、その管理主体毎に保 持されている。ここに述べるメンテナンス機能は、この管理主体を適切に 維持するため、そこに保持されているリソース群を対象として、必要な 処理を施すためのサービスを提供する。
提供される機能には以下のものがある。

  • リソースの探索
    ファシリティ管理機構において、管理対象である各リソースは、利用者の定義する リレーションは勿論であるが、システムとして固有に提供 するリレーションを有している。
    これら特定のリレーションに従ってそのチェインを辿ることにより、
    • 特定のリレーションの保全状態の確認
    • 特定のメンテナンス対象のリソースの探索
    が可能。

  • リソース内部情報の取得と変更
    メンテナンス対象として探索されたリソースに対し、その内部情 報を引き出すことと、その情報を変更することが可能。

  • 変更の反映
    メンテナンス対象としたリソースに対する変更を、管理主体の系に 対して実際に反映することが可能。これら反映は自動的に記録として残される。

  • ロギング情報の参照
    反映情報は自動的にロギングされ、記録(リソースの時間的管理参照)されている。 メンテナンス時にこのロギング情報を参照することが可能。
