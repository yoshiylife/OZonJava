リソースに対する時間的管理について述べる。
ファシリティ管理機構ではその管理対象であるリソースに対し、その生涯を記録する。 具体的には、管理対象であるリソース各々に対する全ての認識できる更 新要求は、それが実際に有効となった時点で、それが自動的に記録される。 記録される情報には以下の種類がある。

  • FMオブジェクトの生成
  • FMオブジェクトの属性の更新
  • FMオブジェクトの管理主体間の移動

これら履歴としての記録は全く当事者のFMオブジェクトには意識されること無く同 等にファシリティ管理機構の管理主体により記録、管理される。
但し、以下の場合のみが若干異なる。

  1. 末尾の「管理主体間の移動」を除き、基本的に当該管理主体に閉じてこれら履 歴は管理、保持される。
  2. 「管理主体間の移動」のみ、管理主体間での情報の伝搬となるため、追 跡可能とする機構を用意しているが、しかしこれは追跡可能性を保証するもの ではない。基本的にリソースに対する時間的な管理は、その責務を管理 主体の管理範囲となる。
