GUI interface

本節では、GUI アプリケーションを実現するためのライブラリを 作成するためのインタフェースを示す。 OZ では、GUI ライブラリとして、基本的な画面構成部品とそのイベントを 取り扱うためのクラス、および基本グラフィックを支援するクラスを 提供する。これには、イメージ処理、ペーストボードなどは含まれない。 ライブラリは、Java の JDK によって配布される AWT クラスライブラリを ベースに、OZ の処理系で要求されるセキュリティ、オブジェクトシステムを 考慮した機能を実装する。

  • ウィンドウ
  • ボタン
  • チェックボックス
  • ラベル
  • メニュー／メニューボックス
  • スクロールバー
  • フィールド
  • イベント
  • グラフィック／フォント
  • その他の画面構成部品
