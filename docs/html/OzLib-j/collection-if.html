Collection interface

青色(イタリック体)のメソッドは、上位クラスから継承したメソッドの再定義。

◇ Plural

すべての集合の共通サービス。


abstract class Plural
  • public abstract Plural clear();

    すべての要素を削除する。

  • public abstract Plural duplicate();

    オブジェクトのシャロウコピー(shallow copy)を取得する。 要素のコピーは行わない。 各具象クラスでは、そのクラスのインスタンスを生成する。

  • public abstract boolean isEmpty();

    要素数が0かどうかを調べる。 0のときtrueを返す。

  • public abstract int size();

    要素数を調べる。 要素数を返す。

  • public abstract Iterator iterator();

    イテレータを取得する。 イテレータのポインタは、その集合の最初の要素を 取り出せる位置に設定される。

◆ Association

キーと値の組。 キーと値は null であってはならない。 値は置き換え可能。


final class Association
  • new create(Root key, Root value);
    オブジェクトを生成する。
    [例外]
    • CollectionException: key または value が null。

  • public String asString();
    オブジェクトの文字列表現を取得する。

  • public Association duplicate();
    オブジェクトのシャロウコピーを取得する。 キーと値のコピーは行わない。

  • public Root key();
    キーを取得する。

  • public Association setValue(Root value);
    値を設定する。 このオブジェクト自身を返す。
    [例外]
    • CollectionException: value が null。

  • public Root value();
    値を取得する。

◆ Dictionary

キーから値への写像。 デフォルトのイテレータは値を返す。 キーを返すイテレータと、キーと値の組(Association)を返す イテレータも取得できる。


class Dictionary : Plural
  • new create();
    オブジェクトを生成する。

  • new create(Comparator forKeys);
    オブジェクトを生成する。 キーどうしの等価比較に forKey を使用する。
    [例外]
    • CollectionException: forKeys が null。

  • new create(Comparator forKeys, Comparator forValues);
    オブジェクトを生成する。 キーどうしの等価比較に forKey を、 値どうしの等価比較には forValues を使用する。
    [例外]
    • CollectionException: forKeys または forValues が null。

  • public Iterator assocIterator();
    イテレータを取得する。 このイテレータは、キーと値の組をクラス Association の インスタンスで返す。

  • public locked String asString ();
    オブジェクトの文字列表現を取得する。

  • public locked Plural clear();
    すべての値を削除する。

  • public locked boolean contains(Root value);
    値 value が登録されているか調べる。 登録されているならば true を返す。
    [例外]
    • CollectionException: value が null。

  • public locked boolean containsKey(Root key);
    キー key で登録された値が存在するか調べる。 存在するならば true を返す。
    [例外]
    • CollectionException: key が null。

  • public locked Dictionary copy(Dictionary dic);
    この Dictionary を別の Dictionary dic のコピーにする。
    [例外]
    • CollectionException: dic が null。

  • public locked Root get(Root key);
    キー key で登録された値を返す。 存在しない場合は null を返す。
    [例外]
    • CollectionException: key が null。

  • public Comparator getComparatorForKeys();
    キーの等価比較を行うオブジェクトを返す。 オブジェクトのコピーは行わない。

  • public Comparator getComparatorForValues();
    値の等価比較を行うオブジェクトを返す。 オブジェクトのコピーは行わない。

  • public boolean isEmpty();
    登録されている値の数が0かどうか調べる。 0ならば true を返す。

  • public Iterator iterator();
    イテレータを取得する。 このイテレータは値を返す。

  • public Iterator keyIterator();
    イテレータを取得する。 このイテレータはキーを返す。

  • public locked Set keys();
    キーの集合を取得する。

  • public Dictionary put(Associaton assoc);
    assoc のキーで値を登録する。
    [例外]
    • CollectionException: assoc またはそのキーか値が null。

  • public locked Dictionary put(Root key, Root value);
    キー key で値 value を登録する。
    [例外]
    • CollectionException: key または value が null。

  • public locked Dictionary remove(Root key);
    キー key で登録した値を削除する。 存在しない場合は何もしない。

  • public int size();
    登録されている要素数を取得する。

  • public locked Set values();
    登録されている値の集合を取得する。

◇ Collection

Dictionary 以外の集合の共通サービス。


abstract class Collection : Plural
  • public abstract boolean contains(Root value);
    値 value を含んでいるか調べる。 含んでいるとき true を返す。

  • public abstract Set difference(Collection c);
    このコレクションに属すが、c には属さない要素の集合を返す。 要素の等価比較には、このオブジェクトの Comparator を使用する。

  • public abstract Set intersect(Collection c);
    このコレクションに属し、かつ c にも属す要素の集合を返す。 要素の等価比較には、このオブジェクトの Comparator を使用する。

  • public abstract Collection remove(Root element);
    element に等しい要素を削除する。 削除する要素数は高々1個。 どれを削除するかは、各具象クラスで異なる。 要素が存在しない場合は、何もしない。

  • public abstract Collection removeAll(Root element);
    element に等しいすべての要素を削除する。 要素の重複が許さないクラスでは、 remove(Root element) と同じ効果を得る。 要素が存在しない場合は、何もしない。

  • public abstract Set union(Collection c);
    このコレクションに属すか、または c に属す要素の集合を返す。 要素の等価比較には、このオブジェクトの Comparator を使用する。

◆ Set

集合。 要素に重複がない。 要素に null を含められない。


class Set : Collection
  • new create();
    オブジェクトを生成する。

  • new create(Comparator comp);
    オブジェクトを生成する。 要素の比較に comp を使用する。
    [例外]
    • CollectionException: comp が null。

  • public locked Set add(Root element);
    element を集合の要素として追加する。
    [例外]
    • CollectionException: element が null。

◇ Sequence

順序付き集合。 集合の中の各要素の位置は、整数(インデックス)で示す。 先頭要素のインデックスは0、それ以外の要素のインデックスは直前の要素の インデックスよりも1大きい。

abstract class Sequence : Collection
  • public abstract Root first();
    先頭の要素を返す。

  • public abstract int indexOf(Root value);
    value と等しい要素のインデックスの中で、最小のものを返す。 存在しなければ、-1を返す。

  • public abstract int indexOf(Root value, int lower);
    value と等しい要素のインデックスの中で、 lower 以上で最小のものを返す。 存在しなければ、-1を返す。

  • public abstract Sequence infix(int start, int end);
    インデックス [start, end) の要素から構成される中置を返す。 要素のコピーは行わない。

  • public abstract Root last();
    最後尾の要素を返す。

  • public abstract Sequence postfix(int start);
    インデックス [start, 要素数) の要素から構成される中置を返す。 要素のコピーは行わない。

  • public abstract Sequence prefix(int end);
    インデックス [0, end) の要素から構成される前置を返す。 要素のコピーは行わない。

  • public abstract int rIndexOf(Root value);
    value に等しい要素のインデックスの中で、最大のものを返す。 存在しなければ、-1を返す。

  • public abstract int rIndexOf(Root value, int upper);
    value と等しい要素のインデックスの中で、 upper 未満で最大のものを返す。 存在しなければ、-1を返す。

◆ Array

配列。 記憶領域の大きさ(登録できる要素数)はユーザが指定する。 要素の追加にはインデックスを指定し、 インデックスによって要素の取り出しを行う。 要素の重複を許す。 要素に null を含められる。

class Array : Sequence
  • new create(int size);
    オブジェクトを生成する。 サイズを size に設定する。
    [例外]
    • CollectionException: size が負。

  • new create(int size, Comparator comp);
    オブジェクトを生成する。 サイズを size に設定する。 要素の比較に comp を使用する。
    [例外]
    • CollectionException: size が負。
    • CollectionException: comp が null。

  • public Root at(int index);
    インデックス index で示される位置の要素を取り出す。 取り出した要素が null の場合もある。
    [例外]
    • CollectionException: index が0未満またはsize()以上。

  • public locked Array clear();
    すべてのインデックスに対する要素を null にする。

  • public Iterator iterator();
    イテレータを返す。 各インデックスに対する要素が登録されているかどうかに関わらず、 ポイントしている要素のインデックスが size()-1 未満であるならば、 hasMoreElements() は true を返す。 イテレータのメソッド atEnd() は、 0 〜 インデックスの最大値(size()-1)に対する要素を ポイントしているときに限り false を返す。

  • public locked Array putAt(int index, Root element);
    インデックス index で示される位置に要素 element を登録する。 既に登録されている要素があっても上書きする。
    [例外]
    • CollectionException: index が0未満またはsize()以上。

  • public locked Collection remove(Root element);
    element に等しい最初の(インデックスが最小の)要素を削除する。 element に等しい要素が存在しないときは何も起こらない。
    [例外]
    • CollectionException: element が null。

  • public locked Collection removeAll(Root element);
    element に等しいすべての要素を削除する。 element に等しい要素が存在しないときは何も起こらない。
    [例外]
    • CollectionException: element が null。

  • public locked Array resize(int newSize);
    サイズを newSize に変更する。 要素は保存される。 ただし、変更前のサイズ(oldSize)と newSize が等しい場合は、 何も起こらない。 また、oldSize よりも newSize が小さい場合は、 0 〜 newSize-1 のインデックスを持つ要素だけが保存される。
    [例外]
    • CollectionException: newSize が負。

◆ SList

単方向リスト。 要素の重複を許す。 要素に null を含められる。


class SList : Sequence
  • new create();
    オブジェクトを生成する。

  • new create(Comparator comp);
    オブジェクトを生成する。 要素の等価比較に comp を使用する。
    [例外]
    • CollectionException: comp が null。

  • public locked SList concatenate(SList list);
    この SList の後ろに list を連結した SList を返す。 要素のコピーは行わない。 この SList 自身と list には何の変更も行わない。

  • public locked SList insertAfter(Root prev, Root element);
    値 prev に等しい最前の要素の次に要素 element を挿入する。
    [例外]
    • CollectionException: prev に等しい要素がない。

  • public locked SList popFront();
    リストの先頭の値を削除する。

  • public locked SList pushBack(Root element);
    リストの最後に要素 element を追加する。

  • public locked SList pushFront(Root element);
    リストの先頭に要素 element を追加する。

  • public locked Collection remove(Root element);
    element に等しい最前の要素を削除する。

  • public locked SList removeAfter(Root prev);
    prev に等しい最前の要素の次の要素を削除する。
    [例外]
    • CollectionException: prev に等しい要素がない。

◇ Iterator

イテレータ。 集合から要素を逐次的に取り出す。 各コレクションのメソッド iterator() は、 Iterator の具象クラスを返す。


abstract class Iterator
  • public abstract boolean atEnd();
    コレクションから取り出せる要素があるかどうか判定する。 次に取り出す要素がないならば true を返す。

  • public boolean hasMoreElements();
    コレクションからまだ要素を取り出せるか調べる。 次に取り出せる要素があるならば true を返す。

  • public abstract Root nextElement();
    コレクションから次の要素を返す。

◇ Comparator

オブジェクトの比較を行う。


abstract class Comparator
  • public abstract boolean compare(Root obj1, Root obj2);
    二つのオブジェクトの比較を行い、 等しいと判定した場合に true を返す。

  • public abstract int hash(Root obj);
    オブジェクトのハッシュ値を返す。
  • public abstract Comparator duplicate();
    オブジェクトのコピーを返す。

◇ StringComparator

文字列の構成文字自体の比較を行う。
Comparator 付き constructor を持つコレクションで文字列を要素にする場合に この StringComparator のインスタンスをコンストラクタの引数に指定する。


class StringComparator : Comparator
  • new create();
    オブジェクトを生成する。

  • public boolean compare(Root str1, Root str2);
    二つの文字列の比較を行い、等しい場合に true を返す。

  • public int hash(Root str);
    文字列のハッシュ値を返す。
  • public Comparator duplicate();
    オブジェクトのコピーを返す。
