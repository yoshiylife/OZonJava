フェデレーション管理

本節では、分散システムにおける動的な構成の変化を可能とするための機構である フェデレーション管理機構の構造仕様を示す。

  • フェデレーションの管理
  • メンテナンス
  • インスタンスのインポート
  • インスタンスのエクスポート
  • フェデレーションの階層的管理
  • フェデレーションの並置的管理
  • ジーナスカテゴリの登録と維持管理
  • インスタンスの生成、登録と維持管理
  • インスタンスへの情報伝搬
  • インスタンスの活性化
  • インスタンスの再構成
  • 論理的リソース管理

制約事項

フェデレーション管理機構には直接は関係ないが、実現のために 以下のクラスを提供する。

  • Authenticator
    セル間通信における認証を組み込むためのクラスである。
    認証システム自体はここでは提供していない。
