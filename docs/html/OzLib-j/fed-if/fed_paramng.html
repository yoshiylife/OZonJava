フェデレーションの並置的管理

フェデレーションの並置的管理とは、Autonomy セル同士が やりとりを行うための仕組みを提供することである。
これを実現するために、Autonomy セルが属する Federation セルを 仲介して、対象 Autonomy セルのタイプを自 Autonomy セルで 変換可能なタイプにみなすための ResourceProxy を取得することにする。

  • public ResourceProxy Autonomy#getAdapter(String resType, Set pivots)
    引数として他の Autonomy セルのタイプおよび中間データタイプの集合をとる。
    当 Autonomy セルが管理している Federation 集合の要素を取り出し、 引数をキーとして、Autonomy#getAdapter(String, String) により 当 Autonomy セルが管理しているアダプタ集合を検索する。
    該当するものがあれば、そのコピーを返す。
    なければ、 null を返す。
