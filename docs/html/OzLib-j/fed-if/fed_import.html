インスタンスのインポート

インスタンスのインポートとは、他の Autonomy セルが保持している リソースの代理オブジェクト ResourceProxy を当 Autonomy セルで 管理することを指す。その目的により、リソースを元に 対応する ResourceProxy を取得するインタフェースを用意する。 一旦取得した情報は、再利用を考え、内部キャッシュに キャッシングするようにする。

  • public ResourceProxy Autonomy#importResource(String resGol, String pivot)
    引数としてリソースの GOL および中間データタイプをとる。
    まず引数のリソースの GOL をキーに使用して、 当 Autonomy セルが管理しているキャッシュを検索する。 該当するものがなければ、 Autonomy#getResourceProxy(String, String, String) により、 新たに ResourceProxy を作成する。
    この際、リソースの GOL から直接作成することが不可能な場合は、 中間データ型を利用することとなる。生成したものは、キャッシュに キャッシングする。最終的に、取得した ResourceProxy を 返すようにする。

  • public ResourceProxy Autonomy#importResource(String gol)
    簡易形式として、このようなメソッドを用意する。
    引数としてリソースの GOL をとる。
    これは、中間データタイプをnullとして、 Autonomy#importResource(String, String) を呼び出すだけである。
