インスタンスの生成、登録と維持管理

ここでいうインスタンスとは、フェデレーションで扱う リソースのことである。扱われるリソースは、すべて FedRoot セルを 継承する。インスタンスの生成は、セルの create(GOL) により 行われ、また Autonomy セルはインスタンスの登録および維持を 行うインタフェースを提供する。

  • public Proxy create(String gol)
    セルのコンストラクタインタフェースである。 引数として、リソースの GOL をとる。

  • public void Autonomy#addResource(String resource)
    引数として、リソースの GOL をとる。
    引数とログレコード ResourceLog を、 Autonomy セルの保持するリソース集合に追加する。

  • public void Autonomy#removeResource(String resource)
    引数として、リソースの GOL をとる。
    引数とログレコード ResourceLog を、 Autonomy セルの保持するリソース集合から取り出す。

  • public boolean Autonomy#containsResource(String resource)
    引数として、リソースの GOL をとる。
    引数とログレコード ResourceLog を、 Autonomy セルの保持するリソース集合から取り出し、 まだリソース集合に保持されていることを判定する。
    判定には、ログレコードのログアウト時間が未設定であるかどうかを 確認する。未設定の時にのみ、真を返す。
