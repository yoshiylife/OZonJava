インスタンスのエクスポート

インスタンスのエクスポートとは、当 Autonomy セルが保持している リソースを他の Autonomy セルで利用する目的で、指定された 中間データタイプに適合した ResourceProxy を返すことを指す。

  • public ResourceProxy Autonomy#exportResource(Set pivots)
    引数として中間データタイプの集合をとる。
    まず引数の中間データタイプの要素を取り出し、 それと当 Autonomy セルのタイプをキーとして、 Autonomy#findAdapter(String, String) により 当 Autonomy セルが管理しているアダプタ集合を検索する。
    該当するものがあれば、そのコピーを返す。
    なければ、 null を返す。
