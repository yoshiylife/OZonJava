インスタンスの活性化

インスタンスの活性化とは、他の Autonomy セルが保持している リソースの代理オブジェクト ResourceProxy を当 Autonomy セルで 利用する際、ResourceProxy 自身が個別の状態を保持する 可能性があることから、コピーを作成して利用することになる。 そのコピーの ResourceProxy を介して、リソースを活用することが できる。これをインスタンスの活性化という。
実際には、コピーの方法はサブクラスによりさまざまであることから、 このインタフェースはサブクラスによりオーバライドされることを 前提とする。

  • public ResourceProxy ResourceProxy#instanciate()
    新たなインスタンスを生成して返す。
    このインタフェースはサブクラスによりオーバライドされることを 前提とする。

  • public ResourceProxy ResourceProxy#instanciate(ResourceProxy prev)
    引数として、連結する ResourceProxy をとる。
    まず、instanciate()を呼び出し、コピーのインスタンスを生成した後、 ResourceProxy#setPrevious(ResourceProxy) により連結を行う。
