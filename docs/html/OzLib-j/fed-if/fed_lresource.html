論理的リソース管理

論理的リソース管理とは、リソースの論理的代替である ResourceProxy の管理を指す。 ResourceProxy は、キャッシュ集合により管理される。
インタフェースとして、キャッシュ集合への追加と キャッシュ集合からの取得を用意する。

  • public void Autonomy#setResourceProxyToCache(ResourceProxy proxy)
    引数として、ResourceProxy オブジェクトをとる。
    キャッシュ集合に、そのオブジェクトを追加する。

  • public ResourceProxy Autonomy#getResourceProxyByCache(String resGol)
    引数としてリソースの GOL をとる。
    キャッシュ集合を検索し、ResourceProxy#getResource() の 戻り値と引数の GOL が一致した最初の要素を返す。
    一致しない場合は、null を返す。
