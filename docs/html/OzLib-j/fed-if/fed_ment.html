メンテナンス

メンテナンスとは、Autonomy セルが保持している ResourceProxy 系の 集合に対する管理を指す。ResourceProxy の情報はインスタンスに 対する参照とそのタイプを保持することとする。また、Autonomy セルの タイプとの比較により、アダプタ、またはサロゲートのどちらかに 分類しそれぞれの集合に対して処理をする必要がある。 あり、管理とは具体的には要素の追加、削除を指す。

メソッド説明

  • public void Federation#addResourceProxy(ResourceProxy proxy)
    引数として ResourceProxy オブジェクトをとる。引数 proxy の オートノミタイプが、当 Autonomy セルの保持しているタイプに 含まれるならばサロゲート集合にインスタンスとリソースタイプを 追加し、そうでなければアダプタ集合に同様に追加する。

  • public void Federation#removeResourceProxy(ResourceProxy proxy)
    引数として ResourceProxy オブジェクトをとる。引数 proxy の オートノミタイプが、当 Autonomy セルの保持しているタイプに 含まれるならばサロゲート集合からインスタンスとリソースタイプを 削除し、そうでなければアダプタ集合から同様に削除する。
