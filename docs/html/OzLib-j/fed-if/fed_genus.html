ジーナスカテゴリの登録と維持管理

ジーナスカテゴリの登録と維持管理とは、ジーナスカテゴリ自身は GenusCategory という抽象セルとして定義されることから、 GenusCategory のサブセルである Autonomy セルと Federation セルの 管理を指す。その管理を支援するインタフェースとして、 カテゴリ名称とカテゴリタイプの管理機能を実装する。

  • public void setCategoryName(String s)
    引数として、カテゴリ名称をとる。引数はそのまま GenusCategory の カテゴリ名称として設定する。

  • public String getCategoryName()
    保持しているカテゴリ名称を返す。

  • public void setType(String s)
    引数として、カテゴリタイプをとる。引数はそのまま GenusCategory の カテゴリタイプとして設定する。

  • public String getCategoryName()
    保持しているカテゴリタイプを返す。
