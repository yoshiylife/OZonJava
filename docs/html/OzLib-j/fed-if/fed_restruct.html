インスタンスの再構成

インスタンスの再構成とは、Autonomy セルが保持している リソースの代理オブジェクト ResourceProxy のキャッシュを 破棄することで再構成する。
破棄のためのインタフェースとして、全ての ResourceProxy を 破棄するものと、指定されたリソースの ResourceProxy のみを 破棄するものを用意する。

  • public void Autonomy#flushCache()
    当 Autonomy セルが保持しているキャッシュ集合を破棄し、 新たなキャッシュ集合を生成し、それを保持することとする。

  • public void Autonomy#flushCache(String resGol)
    引数としてリソースの GOL をとる。
    Autonomy#getResourceProxyByCache(String) により、 対応する ResourceProxy オブジェクトを取得し、 その GOL に対応する ResourceProxy のみをキャッシュ集合から 削除する。
