フェデレーションの管理

フェデレーションの管理とは、 Federation セルが保持している 集合に対する管理を指す。集合の要素は、 Autonomy セルの GOL で あり、管理とは具体的には要素の追加、削除を指す。
また、この機構は Federation セルと Autonomy セルの相互参照に よって実現されるので、追加、削除の処理において両方の集合を 更新する必要がある。

メソッド説明

  • public void Federation#addAutonomy(String autonomy)
    引数として Autonomy セルの GOL をとる。引数 autonomy が、 Federation 内の Autonomy 集合に含まれていなければ、 集合に追加する。 さらに、引数で指定された Autonomy セルの Federation 集合に 当 Federation セルの GOL を Autonomy#attachToFederation(String) により 追加する。この処理は相互に再帰的に行われるので、 デッドロックを避けるために最初に登録状況を判定する必要がある。

  • public void Federation#removeAutonomy(String autonomy)
    引数として Autonomy セルの GOL をとる。引数 autonomy が、 Federation 内の Autonomy 集合に含まれていれば、 集合から削除する。 さらに、引数で指定された Autonomy セルの Federation 集合から 当 Federation セルの GOL を Autonomy#detachFromFederation(String) により 削除する。この処理は相互に再帰的に行われるので、 デッドロックを避けるために最初に登録状況を判定する必要がある。
