インスタンスへの情報伝搬

Autonomy セル間でリソースを利用する場合は、そのポリシーを 変換するための代理オブジェクトを介在する。
この代理オブジェクトは、ResourceProxy オブジェクトにより 実現する。複数の ResourceProxy オブジェクトを連結し、 変換を行うことも可能とする。
実際の変換処理はリソースの種類によりさまざまであることから、 サブクラスによって記述するものとする。

  • public void ResourceProxy#setPrevious(ResourceProxy prev)
    引数として、連結する ResourceProxy オブジェクトをとる。
    このオブジェクトへの参照を保持する。

  • public ResourceProxy ResourceProxy#getPrevious()
    保持している ResourceProxy オブジェクトを返す。

  • public void ResourceProxy#setResource(String resGol)
    引数として、リソースの GOL をとる。
    引数を保持すると同時に、連結された ResourceProxy オブジェクトが ある場合は、そのオブジェクトに対して setResource(String) を 呼び出す。このメソッドは、サブクラスでオーバライドされることを 想定している。

  • public String ResourceProxy#getResource()
    リソースの GOL が保持されていればそれを返す。
    保持されてなく、かつ連結された ResourceProxy オブジェクトが ある場合は、そのオブジェクトに対して getResource() を 呼び出す。このメソッドは、サブクラスでオーバライドされることを 想定している。
