NNTP プロトコル

NNTP クラスとして実現する。

  • 以下のコンストラクタを持つ。
    • create ()

  • インタフェースを以下に示す。

    String article ()
    機能 ARTICLE コマンドをパラメータなしで送信する。
    返り値 ステータスコード

    String article (String msg)
    機能 ARTICLE コマンドを送信する。
    引数 msg メッセージの指定文字列
    返り値 ステータスコード

    String body ()
    機能 BODY コマンドをパラメータなしで送信する。
    返り値 ステータスコード

    String body (String msg)
    機能 BODY コマンドを送信する。
    引数 msg メッセージの指定文字列
    返り値 ステータスコード

    String head ()
    機能 HEAD コマンドをパラメータなしで送信する。
    返り値 ステータスコード

    String head (String msg)
    機能 HEAD コマンドを送信する。
    引数 msg メッセージの指定文字列
    返り値 ステータスコード

    String stat ()
    機能 STAT コマンドをパラメータなしで送信する。
    返り値 ステータスコード

    String STAT (String msg)
    機能 STAT コマンドを送信する。
    引数 msg メッセージの指定文字列
    返り値 ステータスコード

    String group (String group)
    機能 GROUP コマンドを送信する。
    引数 group グループ名
    返り値 ステータスコード

    String help ()
    機能 HELP コマンドを送信する。
    返り値 ステータスコード

    String ihave (String msg)
    機能 IHAVE コマンドを送信する。
    引数 msg メッセージの指定文字列
    返り値 ステータスコード

    String last ()
    機能 LAST コマンドを送信する。
    返り値 ステータスコード

    String list ()
    機能 LIST コマンドを送信する。
    返り値 ステータスコード

    String newgroups (String param)
    機能 NEWGROUPS コマンドを送信する。
    引数 param 条件パラメータ
    返り値 ステータスコード

    String newnews (String param)
    機能 NEWNEWS コマンドを送信する。
    引数 param 条件パラメータ
    返り値 ステータスコード

    String next ()
    機能 NEXT コマンドを送信する。
    返り値 ステータスコード

    String post (String contents)
    機能 POST コマンドを送信する。
    引数 contents ポストする内容
    返り値 ステータスコード

    String quit ()
    機能 QUIT コマンドを送信する。
    返り値 ステータスコード

    String slave ()
    機能 SLAVE コマンドを送信する。
    返り値 ステータスコード
