POP3 プロトコル

POP3 クラスとして実現する。

  • 以下のコンストラクタを持つ。
    • create ()

  • インタフェースを以下に示す。

    String user (String name)
    機能 USER コマンドを送信する。
    引数 name ユーザネーム
    返り値 ステータスコード

    String pass (String str)
    機能 PASS コマンドを送信する。
    引数 str パスワード文字列
    返り値 ステータスコード

    String quit ()
    機能 QUIT コマンドを送信する。
    返り値 ステータスコード

    String stat ()
    機能 STAT コマンドを送信する。
    返り値 ステータスコード

    String list ()
    機能 LIST コマンドを送信する。
    返り値 ステータスコード

    String list (String msg)
    機能 LIST コマンドを送信する。
    引数 msg メッセージ番号の指定文字列
    返り値 ステータスコード

    String retr (String msg)
    機能 RETR コマンドを送信する。
    引数 msg メッセージ番号の指定文字列
    返り値 ステータスコード

    String dele (String msg)
    機能 DELE コマンドを送信する。
    引数 msg メッセージ番号の指定文字列
    返り値 ステータスコード

    String noop ()
    機能 NOOP コマンドを送信する。
    返り値 ステータスコード

    String rset ()
    機能 RSET コマンドを送信する。
    返り値 ステータスコード

    String apop (String name_and_digest)
    機能 APOP コマンドを送信する。
    引数 name_and_digest 名前およびダイジェスト文字列
    返り値 ステータスコード

    String top (String msg)
    機能 ATOP コマンドを送信する。
    引数 msg メッセージ番号および行数
    返り値 ステータスコード

    String uidl (String msg)
    機能 UIDL コマンドを送信する。
    引数 msg メッセージ番号
    返り値 ステータスコード

    String uidl ()
    機能 UIDL コマンドを送信する。
    返り値 ステータスコード
