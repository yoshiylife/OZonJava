SMTP プロトコル

SMTP クラスとして実現する。

  • 以下のコンストラクタを持つ。
    • create ()

  • インタフェースを以下に示す。

    String helo (String domain)
    機能 HELO コマンドを送信する。
    引数 domain ドメイン
    返り値 ステータスコード

    String mail (String from)
    機能 MAIL コマンドを送信する。
    引数 from FROM パラメータ
    返り値 ステータスコード

    String rcpt (String to)
    機能 RCPT コマンドを送信する。
    引数 to TO パラメータ
    返り値 ステータスコード

    String data (String conetents)
    機能 DATA コマンドを送信し、続いて contents の内容を送信する。
    引数 contents 送信するデータ
    返り値 ステータスコード

    String send (String from)
    機能 SEND コマンドを送信する。
    引数 from FROM パラメータ
    返り値 ステータスコード

    String soml (String from)
    機能 SOML コマンドを送信する。
    引数 from FROM パラメータ
    返り値 ステータスコード

    String saml (String from)
    機能 SAML コマンドを送信する。
    引数 from FROM パラメータ
    返り値 ステータスコード

    String rset ()
    機能 RSET コマンドを送信する。
    返り値 ステータスコード

    String vrfy (String user)
    機能 VRFY コマンドを送信する。
    引数 user ユーザ文字列
    返り値 ステータスコード

    String expn (String ml)
    機能 EXPN コマンドを送信する。
    引数 ml メイリングリストアドレス
    返り値 ステータスコード

    String help ()
    機能 HELP コマンドを送信する。
    返り値 ステータスコード

    String help (String str)
    機能 HELP コマンドをパラメータ付きで送信する。
    引数 str コマンド文字列
    返り値 ステータスコード

    String noop ()
    機能 NOOP コマンドを送信する。
    返り値 ステータスコード

    String quit ()
    機能 QUIT コマンドを送信する。
    返り値 ステータスコード

    String turn ()
    機能 TURN コマンドを送信する。
    返り値 ステータスコード
