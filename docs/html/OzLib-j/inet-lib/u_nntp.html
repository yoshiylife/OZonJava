NNTP クラスの使い方

コンストラクタ create によりインスタンスを生成し、そのインスタンスに対してインタフェースとして提供されている各メソッドを呼び出すことにより、NNTP プロトコルで規定される各コマンドを送信することができる。

各メソッドはレスポンスデータ中のステータス文字列を返り値とする。レスポンスデータ全体を得るには、getResponse メソッドを利用する。
