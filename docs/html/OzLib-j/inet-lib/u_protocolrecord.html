ProtocolRecord　クラスの使い方

このクラスはレコード形式、つまり属性ラベルと価の組で表現可能な通信プロトコルの送受信データの処理を簡略化するためのクラスである。

*コンストラクタ

コマンドデータおよびレスポンスデータのそれぞれのフォーマット文字列 と、データの終端文字列を指定する（NetworkClientクラスのコンストラクタを参照）。


*メソッド

-レコードの登録
通常 set メソッドによりラベルと値を登録する。

またコマンド文字列については、ラベル COMMAND が予約されており、setCommand メソッドにより値のみを指定して登録する。


-レコードの値の取り出し
通常ラベルを引数として、get メソッドを呼ぶ。

またレスポンスデータのステータス文字列については、ラベル STATUS が予約されており、getStatus メソッドにより取り出す。


-コマンドデータのエンコード
登録されているレコードデータを、コマンドデータのフォーマットに従ってエンコードした文字列を encode メソッドにより得ることができる。


-レスポンスデータのデコード
レスポンスデータの文字列を引数として decode メソッドを呼び出すことにより、レスポンスデータのフォーマットに従ってデコードし、ステータス文字列等を抽出することができる。



*利用例

このクラスの利用例としては、NetworkClient, NetworkProtocol の各クラスのソースコードを参照のこと。
