FTP プロトコル

FTP クラスとして実現する。FTP プロトコルでは、制御用のコマ ンドおよびその レスポンスの送受信のためのコネクションに加えて、RETR や STOR コマンド等 でデータの送受信を行なうためのコネクションが必要である。またそのデータ用 のコネクションのポートは、クライアント側で listen 状態にある必要がある。 したがって、このコネクションは ServerSocket クラスを利用して実現する。

  • 以下のコンストラクタを持つ。
    • create ()

    String sendCommand (String command)
    機能 指定したコマンド文字列からコマンドデータを生成し、送信する。
    引数 command コマンド文字列
    返り値 ステータスコード
