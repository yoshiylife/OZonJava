ProtocolRecord クラス

このクラスはレコード形式のプロトコルデータを表すためのクラスである。 Dictionary クラスを継承し、属性をキーとした辞書形式によりデータを保持す ることによって、属性値の取り出し等の操作を Dictionary クラスのインタフェースを用いて実現する。

  • ProtcolRecord クラスは以下のコンストラクタを持つ。

    create (String encode_format, String decode_format, String lt)
    引数 encode_format コマンドデータのエンコードに利用するフォーマット文字列
    decode_format レスポンスデータのデコードに利用するフォーマット文字列
    lt データの終端文字列

  • またコマンドデータおよびレスポンスデータのエンコードおよびデコードを、 それぞれ以下のインタフェースによって実現する。

    String encode ()
    返り値 変換後の文字列形式のコマンドデータ

    void decode (String reponse_data)
    引数 reponse_data 受信したレスポンスデータ

  • 属性値に関するインタフェースとして以下のものを持つ。

    String get (String label)
    機能 label で指定された属性の値を返す。
    引数 label 属性文字列
    返り値 属性の値

    String getStatus ()
    機能 ステータスコードの値を返す。
    返り値 ステータスコード

    void set (String label, String value)
    機能 label で指定された属性に値として value を設定する。
    引数 label 属性文字列
    value 属性の値

    void setCommand (String command)
    機能 command で指定された値をコマンド文字列として設定する。
    引数 command コマンド文字列

    String getLineTerminator ()
    機能 データの終端文字列を返す。
    返り値 データの終端文字列
