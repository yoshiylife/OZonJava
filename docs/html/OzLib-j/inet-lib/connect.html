接続の確立/終了

クライアントプログラムからサーバへのコネクションに関するコネクションに関 する処理としては以下のものがある。

  • コネクションの確立
  • コネクションの切断
  • 確立したコネクション上でのデータの送信
  • 確立したコネクション上でのデータの受信
  • コネクション上での異常検出

ところでソケットを利用するためのライブラリがクラスライブラリにより Socket クラスとして提供されている。このクラスは以上の機能をすべて提供し ているため、これを利用してコネクションに関する処理を NetworkClient クラスとして実現する。

  • NetworkClient クラス
  • NetworkProtocol クラス

制約事項
IP プロトコルをベースとするプロトコルのみをサポートする。

利用するサブプログラム

  • ProtocolRecord クラス
  • クラスライブラリ
    • Socket クラス
