インターネットクラスライブラリ

OZ プログラムからインタネット上の各種サービスを利用する場合に、容易にアプリケーションの構築を可能にするためのクラスライブラリを提供する。

このクラスライブラリは以下の機能を持つ。

  • 接続の確立 / 終了
    この機能は、インタネット上の通信プログラムの実現に共通に必要な機能である、OZ 上のクライアントプログラムからサーバに対する接続要求および切断処理のフレームワークを提供するものである。
  • レコード型プロトコル支援
    この機能は、インタネット上の通信プログラムで利用されることの多い、属性と値の組を文字列により表現したレコード型データをやり取りするプロトコルの実装を支援するためのフレームワークを提供するものである。

具体的にプロトコルを実装する対象としては、SMTP (Simple Matil Transfer Protocol)、POP3 (Post Office Protocol Version 3)、FTP (File Transfer Protocol) および NNTP (Network News Transfer Protocol) を取り上げる。

  • SMTP プロトコル
  • POP3 プロトコル
  • FTP プロトコル
  • NNTP プロトコル

利用するサブプログラム

  • NetworkProtocol クラス
  • ProtoclRecord クラス
  • クラスライブラリ
    • ServerSocket クラス
    • InetAddress クラス
