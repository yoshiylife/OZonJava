Thread クラス

  • class Thread : *Root
    スレッドを扱うクラス

    コンストラクタ

    new create(Runnable arg1)
    機能 新しいスレッドを生成する。 生成されたスレッドは、arg1のrun()メソッドを直ちに呼び出す。
    引数
    arg1
    生成されたスレッドの挙動を定めるオブジェクト
    例外 なし

    メソッド

    public void join(long arg1)
    機能 そのスレッドの実行終了を待ち合わせる。 既に終了したスレッドに対するjoin()は、直ちに終了する。
    引数
    arg1
    タイムアウト
    返値 なし
    例外 なし

    public void join()
    機能 そのスレッドの実行終了を待ち合わせる。 既に終了したスレッドに対するjoin()は、直ちに終了する。
    引数 なし
    返値 なし
    例外 なし

    public boolean isAlive()
    機能 そのスレッドがまだ実行中かどうかを調べる。 一般には、実行中だと分かった次の瞬間には実行が停止している可能性があるので、 このメソッドが限られた場合にしか意味を持たないことに注意すべきである。
    引数 なし
    返値
    returncode
    そのスレッドが実行中ならばtrue
    例外 なし

    public void interrupt()
    機能 そのスレッドの実行を中断させる。中断させられたスレッドは、 例外InterruotedExceptionを発生する。
    引数 なし
    返値 なし
    例外 なし

    public boolean isInterrupted()
    機能 自分が中断されたかを返す。
    引数 なし
    返値
    returncode
    中断されていればtrue
    例外 なし

  • class Runnable : *Root
    Runnableは、スレッドの挙動を定めるオブジェクトである。 スレッドは、Runnableをひとつ指定されて生成され、 そのメソッドrun()に記述されているプログラムを実行する。

    コンストラクタ
    コンストラクタはサブクラスで用意する。

    メソッド

    public abstract void run()
    機能 スレッドにより実行されるメソッド。 実行すべきプログラムをサブクラスでオーバライドしなくてはならない。
    引数 なし
    返値 なし
    例外 なし
