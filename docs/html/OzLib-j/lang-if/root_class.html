root システムクラス

  • class Root
    OZのクラスは、すべてRootを継承する必要がある。

    コンストラクタ
    OZ 言語から呼べるコンストラクタはない。

    メソッド

    public final boolean equals(Root arg1)
    機能 1のオブジェクトと当オブジェクトが一致するかを調べる。 値ではなく、オブジェクトの一致をもって等しいとする。 各オブジェクトで再定義することはできない。 コレクションライブラリでは、要素毎のComparatorクラスで用意した compare()メソッドの方を用いる。
    引数
    arg1
    比較するオブジェクト
    返値
    returncode
    一致していればtrue
    例外 なし

    引数 なし
    返値
