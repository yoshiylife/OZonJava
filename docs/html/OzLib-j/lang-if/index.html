Language interface

本節では言語系ライブラリのインタフェースを示す。 OZでは、言語系ライブラリとして以下のものを提供する。 OZ言語で作られるすべてのクラスの上位実装クラスとなるRootクラス。 プログラムの実行単位であるスレッドを扱うためのThreadクラス。 基本データ型をオブジェクトにラップするラッパークラス。 入出力を想定したレコードクラス。 数種のユーティリティクラス。

  • 文字列
  • Thread クラス
  • 基本データ型のラッパー
  • レコード
  • ユーティリティ
