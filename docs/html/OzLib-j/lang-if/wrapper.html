基本データ型のラッパー

  • class Boolean : *Root
    bool型のラッパークラス

    コンストラクタ

    new breed(String arg1)
    機能 文字列表現をbool値に変換した値をもつオブジェクトを生成する。
    引数
    arg1
    "true"ならばbool値true
    例外 なし

    new breed(boolean arg1)
    機能 1の値をもつオブジェクトを生成する。
    引数
    arg1
    bool値
    例外 なし

    メソッド

    public String asString()
    機能 文字列表現を返す。
    引数 なし
    返値
    returncode
    "true"または "false"
    例外 なし

    public boolean booleanValue()
    機能 値を返す。
    引数 なし
    返値
    returncode
    bool値
    例外 なし
