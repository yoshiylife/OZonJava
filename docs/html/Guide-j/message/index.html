コンパイルメッセージ


ここではコンパイル時のエラーメッセージをまとめています。分散統合開発環境( OZ IDE )を利用する自動コンパイル、リリース用コンパイルでは、ここに挙げたメッセージが出力されないので注意してください。



メッセージ中の var_1、 var_2 には、コンパイル中のプログラム中の識別子が出力されます。エラーメッセージ中に識別子が1つの場合にはvar_1 を使用しています。識別子が2つ以上存在する場合には、 var_1、 var_2 と区別しています。


メッセージはアルファベット順に並んでいます。ただし、var で始まるものを1番目に、記号で始まるものを2番目にリストしています。
