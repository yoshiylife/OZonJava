１．Frameクラス

Frameクラスはタイトル付の最上位ウィンドウ（= フレーム）を表現するクラスで、以下のようなメンバを持っています。

コンストラクタメンバ
new newFrame() コンストラクタ。タイトルなしでフレームを生成します。
new newFrame(String arg1) コンストラクタ。タイトルarg1でフレームを生成します。


メソッドメンバ
public void addNotify() フレームのピアを生成します。
public String asString() サイズを示す文字列を答えます。
public void dispose() フレームを破棄します。
public MenuBar getMenuBar() メニューバーを答えます。
public String getTitle() フレームのタイトルを答えます。
public boolean isResizable() フレームの大きさが変更可能かどうかを答えます。
true=変更できます。 false=変更できません。
public void remove(MenuComponent arg1) フレームのメニューコンポーネントarg1を削除します。
public void setIconImage(Image arg1) フレームがアイコン化されたときのイメージデータを
arg1 に設定します。
public void setMenuBar(MenuBar arg1) フレームにメニューバーarg1 を設定します。
public void setResizable(boolean arg1) フレームの大きさが変更可能かどうかを設定します。
true=変更できます。 false=変更できません。
public void setTitle(String arg1) フレームのタイトルを設定します。


OZシステムのフレームは、ウィンドウ右上の をクリックしても終了しません。
フレームを終了させるためには、メニューやボタンを使用して、それがクリックされたときに終了処理を行うようにプログラミングしてください。


２．OZアプリケーションの終了

アプリケーション利用者から終了指示を受け取って終了するようなアプリケーションを構築する場合、いくつか注意することがあります。

  • まず、アプリケーションの終了とは、OZシステムではグローバルオブジェクトの非活性化だということです。したがって、利用者にとってのアプリケーションの終了状態が、次回の起動時（グローバルオブジェクトの活性化）に再現されます。アプリケーションのメニューの設計、メッセージの設計ではこのことに注意してください。

  • グローバルオブジェクトの非活性化を自ら行うメソッドは stopCellメソッドです。このメソッドはOZシステムのあらゆるオブジェクトのメソッドです。グローバルオブジェクトに含まれるいずれかのオブジェクトがこのメソッドを起動すると、グローバルオブジェクトを非活性化する処理が行われます。

  • グローバルオブジェクトの非活性化が行われるとき、そのグローバルオブジェクトの中心となるcell型クラス（= 生成時に動作部クラス識別子が指定されているクラス）で宣言されている quietメソッドがOZシステムによって起動されます。



以下の例は、「はじめてのOZプログラミング」の変形です。MyPrintクラスで stopCell メソッドを起動しています。
stopCellメソッドによってグローバルオブジェクトが非活性化されるときにも quiet メソッドが起動されます。


cell型クラスの宣言 HelloWorld.oz 
    cell HelloWorld    {
        MyPrint obj;

        public void initialize() { /* Do nothing */ }

        public void go() {
            obj=>create();
            for (int i = 0; i < 10; i++) {
                obj->print("Hello world!");
            }
            obj->exit();
        }

        public void quiet() {
            System sys=>create();
            sys->println("Now stopping...");
            flushCell(); }
     }

class型クラスの宣言 MyPrint.oz 
    class MyPrint : *Root {
        System system;

        new create()    {
            system=>create();
        }
    
        public void print(String msg) {
            system->println(msg);
        }
    
        public void exit() {
            stopCell();
        }
    }
