プログラミングガイド


このマニュアルはOZシステムを利用したアプリケーションを作成するために必要な基本的な知識を解説するものです。

OZシステム上で実行されるアプリケーションはOZ言語でプログラミングする必要があります。OZ言語はJavaに似ていますが、より強力なセキュリティ機能とより安全なマルチタスクプログラミングを提供するオブジェクト指向プログラミング言語です。OZシステムはOZ言語のコンパイラを提供しています。
