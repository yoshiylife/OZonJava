入出力操作のプログラミング


ここでは、標準入力、標準出力への入出力操作、ファイルへの入出力操作のプログラミングについて説明しています。

OZでは、このような入出力操作のためのクラスが用意されています。そのクラスはOZ標準クラスライブラリの「入出力ライブラリ」「ユーティリティ」に含まれています。






  • 標準出力とSystemクラス

  • ファイルへの入出力操作

  • バイト単位での入出力ストリーム

  • テキスト入出力ストリーム

  • ソケット通信

  • 外部プロセス起動



入出力操作と関係するSystemクラス以外は、Javaのクラスライブラリと類似のメソッドを用意しています。
