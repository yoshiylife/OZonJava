  2. バイト単位の出力ストリームクラス







１．バイト単位の入力ストリームクラス

abstract class InputStream : *Root すべてのバイト入力ストリームのスーパークラス。
抽象クラス。
class FileInputStream : *InputStream ファイル及びファイル記述子オブジェクトから
バイト単位でデータを読み込むクラス。
class FilterInputStream : *InputStream バイト入力ストリームからデータを読み込んで、
それをフィルタ処理するクラス。
class BufferedInputStream : *FilterInputStream 入力データバッファを設定したバイト入力ストリームから
データを読み込むクラス。
class DataInputStream : *FilterInputStream, DataInput バイト入力ストリームから基本データ型をバイナリ形式で
データを読み込むクラス。
abstract class DataInput : Root データのプリミティブ型をマシンに依存しない方法で
読み込むクラス。抽象クラス。
class PushbackInputStream : *FilterInputStream 1バイトのブッシュバックバッファを提供して、
バイト入力ストリームからデータを読み込むクラス。
class ObjectInputStream : *InputStream オブジェクトデータをバイト入力ストリームから読み込むクラス。
abstract class ObjectInput : DataInput マシンに依存しない方法でオブジェクトデータを読み込むクラス。
抽象クラス。



２．バイト単位の出力ストリームクラス

abstract class OutputStream : *Root すべてのバイト出力ストリームのスーパークラス。
抽象クラス。
class FileOutputStream : *OutputStream ファイル及びファイル記述子オブジェクトへ
バイトデータを書き込むクラス。
class FilterOutputStream : *OutputStream バイト出力ストリームへデータを書き込んで、
それをフィルタ処理するクラス。
class BufferedOutputStream : *FilterOutputStream 出力データバッファを設定したバイト出力ストリームへ
データを書き込むクラス。
class DataOutputStream : *FilterOutputStream, DataOutput バイト出力ストリームへ基本データ型をバイナリ形式で
データを書き込むクラス。
class DataOutput : Root データのプリミティブ型をマシンに依存しない方法で
書き込むクラス。抽象クラス。
class ObjectOutputStream : *OutputStream オブジェクトデータをバイト出力ストリームへ書き込むクラス。
class ObjectOutput : DataOutput マシンに依存しない方法でオブジェクトデータを書き込むクラス。
抽象クラス。
