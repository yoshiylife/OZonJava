テキスト入出力ストリーム


Contents

  1. テキスト入力ストリームクラス







１．テキスト入力ストリームクラス

abstract class Reader : *Root すべてのテキスト入力ストリームのスーパークラス。
抽象クラス。
class BufferedReader : *Reader 入力データバッファを設定したテキスト入力ストリームから
文字データを読み込むクラス。
class LineNumberReader : *BufferedReader データをテキスト入力ストリームから読み込んで、
その行番号を管理するクラス。
class FilterReader : *Reader テキスト入力ストリームからデータを読み込んで、
それをフィルタ処理するクラス。
class PushbackReader : *FilterReader 1文字データのブッシュバックバッファを提供したテキスト入力ストリームから
文字データを読み込むクラス。
class InputStreamReader : *Reader 文字に変換する機能を待ち、バイト入力ストリームから
データを読み込むクラス。
class FileReader : *InputStreamReader ファイル及びファイル記述子オブジェクトから文字データを読み込むクラス。
class PipedReader : *Reader パイプで接続されたテキスト入力ストリームからデータを
読み込むクラス。
class StringReader : *Reader 文字列ソースからデータを読み込むクラス。



テキスト出力ストリームクラス

class Writer : *Root すべてのテキスト出力ストリームのスーパークラス。
抽象クラス。
class BufferedWriter : *Writer 出力データバッファを設定したテキスト出力ストリームへ
文字データを書き込むクラス。
class FilterWriter : *Writer データをテキスト出力ストリームへ書き込んで、
それをフィルタ処理するクラス。
class OutputStreamWriter : *Writer 文字をバイトに変換する機能を持ち、バイト出力ストリームに
データを書き込むクラス。
class FileWriter : *OutputStreamWriter ファイル及びファイル記述子オブジェクトへ文字データを
書き込むクラス。
class PipedWriter : *Writer パイプで接続されたテキスト出力ストリームへデータを
書き込むクラス。
class PrintWriter : *Writer フォーマットされたデータをテキスト出力ストリームに書き込みクラス。
class StringWriter : *Writer 文字列データをテキスト出力ストリームへ書き込むクラス。
