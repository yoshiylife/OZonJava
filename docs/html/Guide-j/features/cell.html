グローバルオブジェクトとGOL、セル


グローバルオブジェクトとGOL

OZシステムではグローバルオブジェクトだけがネットワークを介してコミュニケーションができます。グローバルオブジェクトには、ユニークな識別子を付与します。それをGOL（Global Object Location）といいます。以下の形式でグローバルオブジェクト識別子を付与します。

DNSドメイン名：OZホーム識別子：GOD

GOLはコロン（:）で区切られた３つの要素から構成されます。GODはそのOZホームの利用者が自由に設定することができます。GODには、ドット（.）とアルファベットが使用できます。

たとえば、JP.go.ipa:oz:launcher はランチャのGOLです。
abc.def.ghijklm.no というのは正しいGODです。


グローバルオブジェクトとセル

グローバルオブジェクトを生成すると、そのオブジェクトとコミュニケーションするローカルオブジェクトも生成されます。このグローバルオブジェクト1つと複数のローカルオブジェクトが活動する空間をセルといいます。グローバルオブジェクトの生成によってセルが生成され、グローバルオブジェクトの非活性化でセルが非活性化されます。すなわち、グローバルオブジェクトだけでなく、それとコミュニケーションするローカルオブジェクトもすべて非活性化されます。そして、グローバルオブジェクトを活性化するとセルが活性化され、セルを構成するすべてのオブジェクトが活性化されます。
