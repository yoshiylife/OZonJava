クラス識別子


OZシステムのクラス識別子

OZシステムは以下の形式でクラス識別子を付与してクラスを管理します。 

  DNS形式のドメイン名,OZホーム名,開発者が付ける識別子


たとえば、JP.go.ipa.oz.user.ide.OzIDE_if は OZ IDEのインタフェース識別子、JP.go.ipa.oz.user.ide.OzIDE_cl はクラス識別子です。

末尾が _if の識別子は、インタフェースの識別子です。 _cl はクラス本体の識別子です。

クラス識別子とスクールファイル

クラス識別子は、スクールファイルに記述されるエントリと対応づけられます。
スクールファイルは行単位で以下の情報を記述したエントリから構成されます。 

  "クラス名","インタフェース定義部","動作定義部"

ダブルクォーテーション( " )で囲まれた3つの文字列をコンマで区切っています。最初の文字列がクラス名です。2番目にインタフェース定義部、3番目に動作定義部を指定しますが、これらはOZプログラムで使用されているクラスをユニークに指定するものです。

OzVMとクラス識別子

ＯｚVMでオブジェクトを生成するときには、cell型クラスのクラス識別子(_cl)を指定します。
