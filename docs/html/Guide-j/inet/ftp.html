５．FTPクラスとFTPClientクラスの使用方法

FTPクラスを使用するときには、コンストラクタ create によりインスタンスを生成し、そのインスタンスに対してインタフェースとして提供されている各メソッドを起動することにより、FTP プロトコルで規定される各コマンドを送信することができます。
各メソッドはレスポンスデータ中のステータス文字列を戻り値とします。
レスポンスデータ全体を得るには、getResponse メソッドを利用します。

FTPClientクラスの使用方法も同様です。

使用例 FTPクラス 
    cell FTPTester { 

        public void initialize () { 
        }

        public void go () {
            FTPTesterR run=>create ();
            Thread th=>create (run);
        }

        public void quiet () {
            flushCell ();
        }
    }

    class FTPTesterR : *Runnable {
        new create () {
        }

        public void run () {
            FTP ftp=>create ();
            System sys=>create ();

            sys->println (ftp->connect ("dorothy"));
            sys->println (ftp->getResponse ());
            sys->println (ftp->user ("toto"));
            sys->println (ftp->getResponse ());
            sys->println (ftp->pass ("kansas10"));
            sys->println (ftp->getResponse ());
            sys->println (ftp->nlst ());
            sys->println (ftp->getResponse ());
            sys->println (ftp->getRecvData ());
            sys->println (ftp->quit ());
            sys->println (ftp->getResponse ());
       }
    }
