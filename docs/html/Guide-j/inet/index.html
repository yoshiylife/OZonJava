インターネットアプリケーションの構築


ここでは、既存のインタネットサービスを利用するアプリケーションの構築について説明しています。


OZシステムでは、既存のインタネットサービスを利用するためのクラスライブラリを提供しています。このクラスライブラリを利用するによって、インタネットサービスを利用するアプリケーションを容易に構築することができます。

以下のプロトコルに対するクラスライブラリを用意しています。

  • SMTP (Simple Matil Transfer Protocol)
  • POP3(Post Office Protocol Version 3)
  • FTP (File Transfer Protocol)
  • NNTP (Network NewsTransfer Protocol)


さらに、さまざまなプロトコルに対して汎用的な機能を提供する NetworkClientクラス を提供しています。
このクラスを利用すると上記プロトコル以外のインタネットサービスを利用するアプリケーションの構築を容易なものにすることができます。





  • インターネットクラスライブラリ
  • OZCGI サーバを利用するアプリケーションの構築
