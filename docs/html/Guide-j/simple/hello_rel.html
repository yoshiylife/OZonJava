１．OZ IDEによるリリースコンパイル

OZアプリケーションのリリースはサブジェクト単位で行います。サブジェクトに属するクラスをZIP形式に変換して配布します。配布方法はオフラインでの方法以外に、カタログサーバに登録してオンラインで行う方法があります。

OZ IDE を使用してリリースコンパイルを行うと必要ファイルのZIP形式ファイルが作成できます。
プロジェクトマネージャのCompileメニューから compile for release を選択してください。メッセージウィンドウが開きます。クラス単位でリリースコンパイルが成功したかどうかのメッセージが表示されます。そして、リリースパッケージ（ZIP形式ファイル）の格納場所が標準出力に表示されます。
２．カタログサーバへの登録

リリースコンパイルを行うとカタログサーバにエクスポートすることができます。カタログサーバはアプリケーションやクラスライブラリをカタログするサーバです。カタログサーバからランチャを使ってアプリケーションを実行することができます。（詳細はカタログサーバのマニュアルを参照してください。）
