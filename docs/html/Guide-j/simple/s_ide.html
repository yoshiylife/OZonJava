統合開発環境を利用したアプリケーション開発


ここでは、アプリケーションを開発しそれをリリースするまでのプロセスとそのプロセスの中で発生するデバックについて説明しています。OZ統合開発環境（以下、OZ IDE と略称しています。）をアプリケーション開発で利用する方法についても説明しています。「Helloプログラムとその実行」を読んでいただいていることを想定して話を進めています。

課題として、Helloという文字列をウィンドウに表示するアプリケーションを構築します。

（ここで扱うプログラムは、環境変数OZHOMEのサブディレクトリuserのしたにsampleという名前で格納されているものを簡略化のために一部修正しています。）
