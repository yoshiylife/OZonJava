はじめてのOZアプリケーション


ここではOZシステムを利用するアプリケーションの構築の初歩として簡単なプログラムを紹介し、それをコンパイル、実行する方法を説明しています。

さらに、OZシステムが提供している統合開発環境を利用してコンパイルをする方法をご紹介します。


次に示す Hello.oz がここで説明をするもっとも簡単なOZプログラムです。


OZプログラム Hello.oz 

    cell Hello{
        public void initialize(){;}

        public void go(){
            println("Hello!");
        }
         
        public void println(String str){
            System system=>create();  
            system->println(str);
        }

        protected void quiet(){
            flushCell();
        }
    }






以下の「Helloプログラムとその実行」では、OZ言語によるプログラミングの基礎的事項を説明しています。「統合開発環境を利用したアプリケーション開発」では、Helloプログラムをアプリケーションとしてリリースするまでの手順を説明しています。
