OZソースファイルのコンパイル


ここではソースプログラムのコンパイルについて統括して説明しています。

OZプログラムのコンパイルは3つの方法で行うことができます。

コマンドプロンプトからOZコンパイラを起動する方法
オペレーティングシステムが提供している標準のコマンドプロンプトからOZコンパイラを起動して、インタフェース部のコンパイルおよび動作部のコンパイルを行うことができます。


統合開発環境からコンパイルする方法
コマンドプロンプトからOZコンパイラを起動して行うコンパイルに加えて、自動コンパイルやリリースパッケージ作成コンパイルを行うことができます。


既存環境インタフェース機能を利用するコンパイル
このコンパイルは2通りの方法で利用することができます。
  1. 分散環境で個別に開発したクラスを1つの環境下で統合する
  2. OZシステムを導入していないマシンからOZプログラムをコンパイルする


コンパイルメッセージについては「コンパイルメッセージ」にまとめています。
