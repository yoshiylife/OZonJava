既存環境インタフェース機能を利用するコンパイル


OZ既存環境インタフェース機能を利用するを利用すると、ネットワークでつながったリモートマシン上でコンパイルを行うことができ、そのコンパイルの出力である実行形式ファイルがリモートマシンに格納されます。

既存環境インタフェース機能とは、WWWクライアントからCGIを経由して渡されるOZシステムの情報を解釈し、リモートコンパイル、リモート実行、カタログ閲覧などアプリケーションの構築と実行に必要なサービスを提供するものです。

既存環境インタフェース機能のコンパイルは、OZHOME/lib/inet ディレクトリにある wwps.html ファイルを適切に編集して行います。



既存環境インタフェース機能の詳細については、「ツールリファレンス」のワールドワイドプログラミングサーバを参照してください。
