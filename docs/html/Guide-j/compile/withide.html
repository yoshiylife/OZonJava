統合開発環境からのコンパイル


ここでは、統合開発環境からのコンパイルの概要について説明しています。



OZ統合開発環境を用いると、以下のようなコンパイルを行うことができます。

  1. インタフェースコンパイル
  2. コンパイル
  3. サブジェクトコンパイル
  4. 修正個所自動コンパイル
  5. リリース用コンパイル


さらに、コンパイルしたプログラムのテスト実行を、OZ統合開発環境から行うことができます。
