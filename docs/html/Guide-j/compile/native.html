コマンドプロンプトからのコンパイル


ここでは、OZコンパイラのコマンドラインの起動とコマンドライン引数について説明しています。



OZコンパイラは、入力パラメータとして、

  • ソースファイル
  • スクール

の二つが必要です。

ソースファイルは、複数を同時に指定することができます。



コマンド

ozc [オプション] -school school.s source1.oz [ source2.oz .....]
  • -school の代わりに -s を指定することもできます。



オプション
-school <file> または -s <file>

スクールファイルを指定します。


-log <file>

OZコンパイラの画面出力の出力先をファイルに変更します。デフォルトはプラットフォーム環境の標準エラー出力です。


-interface

インタフェース部のみのコンパイルを実行する。


-verbose または -v

コンパイルの実行経過を表示します。


-version

バージョン情報を表示する。


-help または -h

コンパイルオプションの簡単な説明を表示します。


-domain または -d

クラスIDの先頭に付加するJavaのパッケージのドメイン名を指定する。



エラー

1つのスクールファイルと、1つ以上のソースファイル（= 拡張子が.ozのファイル）が指定されないとエラーになります
また、不正なプログラムコードが記述されている場合には、コンパイルメッセージを表示し、処理を中断します。


例: コマンドプロンプトからのOZコンパイルの実行 

＞ ozc -s test.s Test.oz
