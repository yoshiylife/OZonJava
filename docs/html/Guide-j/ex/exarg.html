OzVMの起動とそのオプション


ここでは、OzVMのコマンドラインからの起動とコマンドライン引数について説明しています。



OzVMはGOLを指定して起動すると、そのGOLが指定するグローバルオブジェクトを活性化し、そのグローバルオブジェクトとコミュニケーションして実行管理します。インタラクティブモードで起動すると、複数のグローバルオブジェクトを活性化して実行管理することができ、デバックのための出力を行うことも可能です。



コマンド

oz GOD GODで指定されるグローバルオブジェクトを起動します
oz [オプション]



オプション
-i
インタラクティヴモード。
コマンド行入力を促すプロンプトを出力します。

-s
標準入力からOzVMコマンドを読み込んで実行します。
標準入力からのコマンドの読込みができなくなると、直ちに終了します。

-q
実行中に例外が発生した場合、通常の終了手続きを行ってOzVMを終了します。
このオプションが指定されていない場合には、コマンドの実行で例外が発生してもOzVMは終了ません。

-e
直ちにOzVMを終了する場合に、通常の終了手続きを行わずにOzVMを終了します。
このオプションが指定されると、グローバルオブジェクトの非活性化処理を行わず強制終了します。

-f <コマンドファイル名>
<コマンドファイル名>で指定したファイルからOzVMコマンドを読み込んで実行します。
読み込むコマンドがなくなった後、活性化されているグローバルオブジェクトがなくなった時点でOzVMを終了します。
ファイルが指定されない場合には Not Found file: oz というメッセージを出力します。
OzVMコマンド以外を含むと、その行の実行でUnknown command: xxxx というメッセージを出力します。

-v
実行履歴、発生した事象の情報を表示します。
