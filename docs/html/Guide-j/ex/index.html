OZアプリケーション実行環境


OZアプリケーションはOzVM上で稼動します。OZシステムが提供するさまざまなツールもOzVM上で稼動します。ここでは、OzVMの機能を概略し、OzVMシェルの使用方法、実行ログをご紹介しています。


OzVM機能概略

OzVMは以下の機能を備えています。

  • プロセス管理機能
  • グローバルオブジェクト管理機能
  • クラスローダ機能
  • OzVMシェル
  • 実行ログの作成


クラスローダ機能とは、実行時に必要となるクラスをロードする機能です。必要に応じて、ローカルマシンのクラス、および、ネットワークを介したマシンからクラスをロードすることができます。OZシステムではネットワーク上でクラスを共有させることが可能なのです。
