OzVM実行ログ


ここでは、OzVM実行ログについて説明しています。



OzVMを起動すると必ずログファイルが作成されます。コマンドプロンプトで、OzVMを起動するためのコマンド
oz を起動すると最初に表示されるメッセージがログファイル名です。

OzVMの起動とログファイル名 

    > oz -f hello.ins
    Logging file F:\oz\logs\dorothy01080.log
    ...


ログファイルには、OzVMの出力するメッセージが表示されます。

ログファイル 

Fri Feb 27 08:41:49 OZ Executor start on Java 1.1.4
Fri Feb 27 08:41:49 ExClassLoader#ExClassLoader: oz.classloader.deliveryAgent.url is null
Fri Feb 27 08:41:50 ExSecurityManager#greeting: ExSecurityManager is ready.
Fri Feb 27 08:41:53 ***** SHUTDOWN: CellManager *****
Fri Feb 27 08:41:53 ***** SHUTDOWN: CommManager *****
Fri Feb 27 08:41:54 OZ Executor exit(0)


世界標準時で事象が起こった日時が表示されます。

アプリケーションがログファイルにメッセージ出力することもできます。

アプリケーション実行中に異常が発生した場合、実行時エラーメッセージも出力されます。
