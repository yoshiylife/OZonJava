OzVMによる認証システムの選択

マシン myhost 上にある、OZ ホーム ME 中の セル A から、 マシン yourhost 上にある、 OZ ホーム YOU 中の セル B への リモートメソッド起動で、実際に使用される認証システムは、 以下の条件をすべて満たすものである。

  • ME/etc/encoders/myhost.cipher に含まれている。
  • YOU/etc/encoders/yourhost.cipher に含まれている。
  • ME/A/.cipher がある場合、そこに含まれている。
  • YOU/B/.cipher がある場合、そこに含まれている。
以下の条件がすべて満たされる場合は、 認証システムを使わない、認証なしの通信が行われる。
  • ME/A/.cipher がないか、 PlainText が含まれている。
  • YOU/B/.cipher がないか、 PlainText が含まれている。
(認証なしの場合を含めて) 複数の認証システムが使用される可能性がある場合、 どれが使われるかは、OzVMの実装に依存し、ここでは規定されない。
