認証システムリストファイルの書式

$OZHOME/etc/encoders/myhost.cipher ファイルと GOD/.cipher ファイルをまとめて、 認証システムリストファイルと呼ぶ。 認証システムリストファイルの書式は、以下の通りである。

  • 一行にひとつのレコードを含む。
  • ひとつのレコードは、認証システム名か、または、 PlainText という文字列 (大文字小文字もこの通りでなければならない) である。
  • 認証システム名とは、 その認証システムの認証システムクラスの fully qualified クラス名である。
  • 認証システムクラスとは、 クラス JP.go.ipa.oz.system.OzCipher の サブクラスであって、 その認証システムに対するインタフェースを提供するクラスである。 認証システムクラスは、 その認証サーバ OZ から利用可能にしようとする者が、 OzCipher の提供する SPI を実装して用意する。
    fully qualified クラス名とは、パッケージ名部分を含んだ、 完全な Java のクラス名である。
  • PlainText という文字列は、 そのセルが平文の通信も許していることを示す。 すべてのマシンは、常に平文の通信が可能なので、 $OZHOME/etc/encoders/myhost.cipher ファイルには PlainText というエントリがなくても、 あるのと同様に扱われる。
