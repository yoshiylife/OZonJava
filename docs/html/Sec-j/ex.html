認証システムを利用するための設定
マシンごとの設定
マシン myhost が使用可能な認証システムを、 $OZHOME/etc/encoders/myhost.cipher ファイルに列挙する。 "myhost" は、ドメイン名を含まない、ホスト名のみの表記とする。
セルごとの設定
そのセルが利用してよい認証システムを、 GOD/.cipher ファイルに列挙する。 GOD は、そのセルのグローバルオブジェクトディレクトリである。
設定の変更方法
以下を順に実行することで設定を変更できる。
  1. 設定ファイル ( $OZHOME/etc/encoders/machine.cipher または GOD/.cipher ) を編集する。
  2. 変更を反映させたいセルの キャッシュファイル GOD/.ciphercache を削除する。
  3. そのセルを一旦停止させてから、再起動させる。
