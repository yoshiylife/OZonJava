サービス要求の作成(サービス提供者/ユーザ)

  • ブローカ作成/登録、サービス要求クラスの作成
    サービス提供者は、サービスに応じたサービス要求クラス UserServiceRequest を作成し、コンポーネントカタログに登録する。
    サービス利用者は UserServiceRequest の必須条件、優先条件を 書き換え、必要なサービス要求を作成する。

  • サービス要求クラスの作成
    サービス提供者は、該当サービスのサービス要求クラス ServiceRequest を 継承して自身の要求を必須条件、優先条件として記述したクラス UserServiceRequest を作成する。

  • UserServiceRequest の例
    class UserServiceRequest :*ServiceRequest{
  new create(){super=>create();}
  //必須条件
  public bool requirement(UserBroker br){
    return br->keyWordSearch();
  }
  //優先条件
  public int preference(UserBroker br){
    int i,j;
    if(br->wholeText())i=1;else i=0;
    if(br->japanese())j=1;else j=0;
    return i*2+j;
  }
}
