トレーダの起動/設定

  • 起動方法
    ランチャを用いて $OZHOME/objects/trader を起動する。 トレーダの起動個数は、特に制限はないが 検索エンジンであるので、自治的な管理ドメインに一つ 程度が望ましい。

  • CoWorkingTrader の設定
    CoWorkingTrader は、検索を委譲する他のトレーダで、 以下のように設定する。 
     >addCoTrader 検索委譲する他のトレーダのGOL
