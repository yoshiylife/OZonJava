概要

トレーダは、OZ のセルで提供されるサービスをOZプログラムから検索する 検索エンジンである。
ユーザはサービス要求オブジェクトをトレーダに投じることによって 柔軟にサービスを検索し、該当サービスを提供するセルの Proxy を 取得することができる。
サービス提供者は、サービスの Proxy を発行できるブローカ(Broker) という仲介オブジェクトを作成し、これをトレーダに登録する。
サービス利用者は、取得したいサービスを検索するための サービス要求(ServiceRequest)を作成し、トレーダへ検索依頼する ことにより、該当サービスへの Proxy を取得する。 このサービス要求はサービス名に加えて、必須条件、優先条件を メソッドで記述できるため柔軟な検索が可能である。
